Seurat v5 -Combine multiple samples, normalization, PCA, clustering and visualization

Description

This tool can be used to perform a combined analysis by merging or integrating multiple data sets. The combined analysis includes similar steps as the single sample analysis which includes normalization, PCA, clustering and visualization. Use this tool instead of the single sample analysis tool when you want to analyze multiple samples at the same time.

Parameters

  • Number of variable genes to return in SCTransform [3000]
  • Combining method [merge]
  • Number of PCs to compute [50]
  • Print loadings in a file [no]
  • Number of genes to list in the loadings file [5]
  • Dimensions of reduction to use as input [30]
  • Resolution for granularity for clustering [0.8]

Details

As inputs, give two or more Seurat objects after the filtering step. This tool will perform similar analysis for multiple samples as explained in the tool Seurat v5 -Normalization, PCA, clustering, and visualization.

Merge vs integration as the combining method

The "combining method" parameter determines which method is used to combine. By default, merge is used, but the user can also choose to use integration. Merge simply combines the objects into one.

The integration method uses canonical correlation analysis (CCA) to identify common sources of variation across datasets. After this, the tool identifies anchors, generates scores (=weights) for these anchors and then uses the anchors to integrate the datasets. The integration method should be used to remove strong batch effects present in the data. More information on data integration can be found from single-cell tool Seurat v5 -Integrate multiple samples.

Further information

For more details, please check the Seurat tutorial and Introduction to scRNA-seq integration.

Output

  • seurat_obj_multiple.Robj: The Seurat R-object to pass to the next Seurat tool, or to import into R. You cannot open this file in Chipster but you can view the contents of the object using the Seurat v5 -Extract information from Seurat object tool.
  • merged_plot.pdf: If the data is combined by merging, this tool will output a UMAP and SpatialDimPlot where the data has been merged.
  • integrated_plot.pdf: If the data is combined using integration, this tool will output a UMAP and SpatialDimPlot where the data has been integrated.
  • PCAloadings.txt: If selected, the loadings (=genes responsible) of PCs.
