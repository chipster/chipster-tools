Create HISAT2 index

Description

This tool creates index files for the HISAT2 aligner.

Parameters

None

Input

  • input.fa: reference genome FASTA file
  • input.gtf: reference genome GTF file
If the input.gtf is given, the tools tries to create an index with splice site and exon annotations, which requires huge amount of memory for larger genomes. Memory consumption is much lower if the GTF file is not given.

Output

This tool produces a Tar package that can be given to the tools "HISAT2 for single end reads and own genome" and "HISAT2 for paired end reads and own genome".

Reference

This tool is based on the HISAT2 package. Please cite the following article: Kim D, Langmead B and Salzberg SL. HISAT: a fast spliced aligner with low memory requirements Nature Methods 2015.
