Seurat v5 -Visualize gene expression

Description

With this tool, the user can visualize selected genes using a spatial feature plot. As input, the user can provide the Seurat object (.Robj) and the name of the gene(s) of interest (e.g., Hpca, Ttr). For listing multiple gene names, a comma (,) should be used as a separator.

Please note that genes given as input should be given exactly as they are named in the Seurat object. The tool will only visualize genes that are found in the Seurat object. You can use the Seurat v5 -Extract information from Seurat object tool to see the genes in your Seurat object.

Parameters

  • Gene name ["Hpca, Ttr"]
  • Point size [1.6]
  • Minimum transparency of the points [1]
  • Maximum transparency of the points [1]
  • Determine color scale based on all genes [no]

Details

Spatial feature plot shows molecular data overlaid on top of the tissue image. Highly expressed marker genes will be clearly displayed in the image.

Note that you can select whether the color scale is based on all genes or individual genes. By default, the color scale is determined for each gene individually and may differ between genes. If you wish to compare gene expression between different genes, it is useful to set this parameter to "yes" so that the color scale is the same for all genes.

For more details on the spatial transcriptomics analysis pipeline, please check the Seurat tutorial.

Output
