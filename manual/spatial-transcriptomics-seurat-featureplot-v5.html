Seurat v5 -Visualize gene expression

Description

With this tool, the user can visualize selected genes using a Spatial feature plot. As input, the user can provide the Seurat R-object (.Robj) and the name of the gene(s) of interest (e.g., Hpca, Ttr).

Please note that genes given as input are case-sensitive so they should be given as input exactly as they are named in the Seurat object. If you list multiple gene names, use a comma (,) as a separator. You can use the Seurat v5 -Extract information from Seurat object tool to see the genes in your Seurat object.

Parameters

  • Gene name ["Hpca, Ttr"]
  • Point size [1.6]
  • Minimum transparency of the points [1]
  • Maximum transparency of the points [1]

Details

Spatial feature plot shows molecular data overlaid on top of the histological tissue. Highly expressed marker genes will be clearly displayed in the histological image.

For more details, please check the Seurat tutorial.

Output

  • Feature_plot.pdf
