Seurat -Remove clusters

Description

You can use this tool to remove clusters from your data. You can either list the clusters you would like to include or the clusters you would like to remove.

Parameters

  • Select whether to include or remove clusters [remove]
  • Select names of clusters to include or remove [3]

Details

Please input the Seurat R-object (Robj) after the clustering step.

If you provide cluster names not present in the data, the tool will generate an error.

For more details, please refer to the Seurat tutorials.

Output

  • seurat_obj.Robj: The Seurat R-object to pass to the next Seurat tool, or to import into R, but it is not viewable in Chipster.
  • ClusterPlots.pdf: Cluster plots where you can confirm that correct clusters were included or removed.
