Utilities/ Annotate Ensembl identifiers

Description

Annotates Ensembl identifiers in the input table with gene symbols (external gene names) and descriptions.

Parameters

None

Details

Ensembl REST API is used to fetch the annotations. Ensembl IDs need to be in the first column or as row names in the input file.

Maximum of 100 000 Ensembl IDs can be annotated in one job.

Output

As an output you get a table with the annotated IDs and the original values from the input table.

References

This tools uses Ensembl REST API to provide Ensembl data, please cite Yates et. al. (doi:10.1093/bioinformatics/btu613) and Zerbino et. al. (doi.org/10.1093/nar/gkx1098).
