Annotate Ensembl identifiers

Description

Annotates Ensembl identifiers in the input table with gene symbols and descriptions. Please note that annotation should be done after other analysis steps. This is because the description column contains sometimes special characters, which can confuse the R/Python code of the analysis tools.

Parameters

None

Details

Ensembl REST API is used to fetch the annotations. Ensembl IDs can be in the first column or as row names in the input file. They can also be in the middle of the file if the column title is ensembl_id.

Maximum of 100 000 Ensembl IDs can be annotated in one job.

Output

As an output you get a table with the annotated IDs and the original values from the input table.

References

This tools uses Ensembl REST API to provide Ensembl data, please cite Yates et. al. (doi:10.1093/bioinformatics/btu613) and Zerbino et. al. (doi.org/10.1093/nar/gkx1098).
