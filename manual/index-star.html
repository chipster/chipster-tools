Create STAR index

Description

This tool creates index files for the Star aligner.

Parameters

None

Input

The genome's fasta file is taken as an input.

Output

This tool produces multiple files that together form the index of the given genome.

References

This tool uses the STAR aligner. Please cite the article:

Alexander Dobin et al: STAR: ultrafast universal RNA-seq aligner (2013) Bioinformatics 29: 15-21.

Please see the STAR manual for more details.
