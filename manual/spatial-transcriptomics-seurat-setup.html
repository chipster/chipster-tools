Seurat v4-Setup and QC

Description

This tool creates a Seurat object of 10x Visium spatial data. After setting up the R-object (.Robj), some quality control plots are drawn.

Parameters

  • Sample or group name [empty]

Details

This tool creates a Seurat object from a tar packed folder containing the following 10X Genomics output files:

  1. filtered_feature_bc_matrix.h5
  2. aggregation.csv
  3. aggr_tissue_positions_list.csv
  4. scalefactors_json.json
  5. tissue_hires_image.png
  6. tissue_lowres_image.png

These six files are generated by the Space Ranger -pipeline. here. Make sure that you use the filtered data which contains only the cell barcodes present in your data, as opposed to the unfiltered data containing millions of barcodes (this will very likely jam the tools).
You need to tar the folder into a package. There are graphical programs for this, such as 7-zip, or you can use command prompt (Windows) / Terminal (Mac) with command: tar -czvf name-for-your-package.tar.gz /path/to/folder-you-want-to-tar. You can also use the Chipster tool Utilities - Make a tar package.

This tool also produces the following quality control plots (note that it might be easier to view the large QCplots.pdf file with an external viewer):
  • Violin plots showing the number of UMIs (nCount_Spatial) in the cells.These plots can be used to estimate the upper limits for number of genes or molecules per cell and the percentage of mitochondrial transcripts, which you can use in the filtering tool.
  • Spatial feature plot showing number of UMIs as spots over the spatial image .

For more details, please check the Seurat tutorial.

Output

  • seurat_spatial_setup.Robj: The Seurat R-object to pass to the next Seurat tool, or to import to R. Not viewable in Chipster.
  • QCplots.pdf: Quality control plots
