Filter by column value

Description

Filters rows in a table based on the numeric values in the selected data column.

Parameters

  • Column to filter by (...) [empty]
  • Cutoff (-100000...100000) [1]
  • Filtering criteria (equal-to, smaller-than, larger-than, within, outside) [smaller-than]
  • Does the first column have a title (yes, no) [yes]

    Details

    Filters rows in a table based on continuous values in the selected data column. The options "within" and "outside" can be used to filter based on a symmetrical range of values. For example, setting the cutoff to 2 and the criteria to outside will give rows where the value is larger than 2 or smaller than -2.

    Output

    A tabular text file containing only the rows that pass the filtering criteria.
