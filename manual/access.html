If you would like to use Chipster to analyze your data, you have three options:

  1. Setting up a Chipster server at your institute. This is free of charge, but you need an IT person to help you and suitable computing resources to run the server. The actual installation is easy, because Chipster is available as a ready-to-use virtual machine image. If you have an IT support person but lack suitable computing resources, you can install and run the Chipster server free of charge in the EGI Federated cloud.

  2. Use our Chipster server at CSC in Finland. This is free of charge for users from Finnish universities and government research institutes (for open research), but others need to pay for the computing time. However, free 3-week evaluation accounts are available for everybody. Please see below how you can get a user account.
    • Users from Finnish universities
      • You can log in with your Haka or Virtu account or use a CSC user account. CSC user account is free and can be obtained using myCSC customer portal. Read more about creating an CSC account in CSC's user guide at docs.csc.fi. Users who already have a regular CSC username and password can use those for Chipster.
    • Users from other universities and non-profit organizations
      • Free 3-week evaluation accounts can be obtained by filling by sending email to chipster@csc.fi. Please see the instructions.
      • Contact us if you want to purchase a user account for academic use. One year personal single user account costs 500 euros.
    • Corporate users
      • Both personal single user accounts as well as company accounts are available. Please contact us for a quote.
    • Training use
      • We also provide training accounts to our Chipster server in Finland. Note, that users from Finnish universities and research institutes can use their HAKA and VIRTU logins to access Chipster, so training accounts are not necessary. However, we kindly ask you to inform us about your upcoming Chipster courses (number of participants, type of analysis jobs) so that we can add computing resources to the server if needed. Please email us (chipster (at) csc.fi) about your training needs. More information about using Chipster and related materials in training.

  3. Use Chipster in the EGI Applications on Demand Service. This is free of charge, but you need register for an EGI account (please see the instructions).

Requesting evaluation account for the CSC Chipster service

To get an evaluation user account for the CSC Chipster service, please send email to chipster@csc.fi with the following information. As the subject of the email please use Chipster evaluation account and in the body of the email include the following inofrmation:

  • First name
  • Last name
  • Position
  • Institute
  • Email

Please also see the Chipster privacy notice.
