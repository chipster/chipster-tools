Proportional prevalence filtering

Description

Filters out any OTUs below a user-specified prevalence threshold (%) using phyloseq. The resulting phyloseq object is saved in Rda format.

Parameters

  • Prevalence threshold (%) for filtering (accepts integer values from 1 to 25; the default is 5)

Details

The tool requires the following input files:

  • A phyloseq object in Rda format
  • Data groupings are automatically fetched from a phenodata file linked to a Mothur shared file (file.opti_mcc.shared)
The tool employs the following R functions and packages:
  • prune_taxa() (phyloseq)
For details on the steps used by this tool, see Callahan et al. (2016).

Output

The tool produces the following output files:

  • Phyloseq object following prevalence filtering (ps_prevfilter.Rda)
