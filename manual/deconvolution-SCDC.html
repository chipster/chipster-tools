Deconvolution Tool (SCDC)

Description

This tool estimates the abundance (or proportion) of different celltypes in Visium spots using a single cell reference dataset.

Parameters

  • Subset of clusters ['L4']
  • Top genes [20]

Details

Please input the Seurat R-object (Robj) and single-cell RNA-sequencing (scRNA-seq) reference dataset. (Please be mindful of the order that you click each object, as the Seurat object should be first and the reference dataset should be second.)

The input Seurat object may first need to be generated using the tool: Seurat v4 -Subset out anatomical regions based on clusters. To select clusters present in the data, you can refer to the third plot generated using the tool Seurat v4 -PCA, clustering, and visualisation. This plot shows all of the the color-coded clusters which have been mapped to different anatomical regions of the tissue slice.

If you would like more information about the SCDC tool, please refer to the following two resources:

  • NBIS (National Bioinformatics Infrastructure Sweden) tutorial on the deconvolution of spatially resolved transcriptomics (SRT) data

  • SCDC: Bulk Gene Expression Deconvolution by Multiple Single-Cell RNA Sequencing References vignette, written by its own developer, Meichen Dong.

Outputs

  • seurat_obj_deconv.Robj: This Seurat R-object now includes a new SCDC assay, and this can be imported into R. However, this is not viewable in Chipster.

  • SpatialFeaturePlot.pdf: Plot(s) are generated for each selected celltype. Different colored spots are overlayed on top of the tissue slice. The colors represent the predicted proportions of that particular celltype in that geographic area of the tissue slice.

  • SpatialPlot.pdf: Four tissue slice plots with corresponding colored spots are generated -- one for each of the top four celltypes.

  • VlnPlot.pdf: Four violin plots are generated -- one for each of the top four celltypes. The x-axis shows the selected clusters, each designated by its corresponding number, and the y-axis shows each cluster's expression level.
