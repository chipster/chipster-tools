Create DEXSeq index

Description

This tool convertes a gtf file for DEXSeq.

Parameters

None

Input

The genome's gtf file is taken as an input.

Output

This tool produces a gtf file for DEXSeq.

References

Anders S, Reyes A, Huber W. Detecting differential usage of exons from RNA-seq data. Genome Res. 2012 Sep 5.
