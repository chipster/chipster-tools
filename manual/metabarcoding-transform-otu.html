Transform OTU counts

Description

This tool is used to transform raw OTU counts within a phyloseq object, after which the data are saved in Rda format. Four options are available:

  • CLR transformation (with pseudocount)
  • Relative abundance (%) conversion
  • Hellinger transformation
  • DESeq2 format conversion and variance-stabilizing transformation (VST)

Parameters

  • Choice between data transformation types [CLR, relative abundance (%), Hellinger transformation, DESeq2 format conversion and VST; default is CLR]
  • Phenodata variable used to specify the experimental design (when converting data to the DESeq2 format)

Details

The tool requires the following input files:

  • A phyloseq object in Rda format
  • Data groupings are automatically fetched from a phenodata file linked to a Mothur shared file (file.opti_mcc.shared)
The tool employs the following R functions and packages:
  • transform() (microbiome)
  • phyloseq_to_deseq2() and otu_table() (phyloseq)
  • estimateSizeFactors(), estimateDispersions() and getVarianceStabilizedData() (DESeq2)
Technical notes:
  • The CLR transformation applies a pseudocount of min(relative abundance)/2 to zero relative abundance entries in the OTU table (see microbiome for details).
  • Converting data to into the DESeq2 format requires a user-specified phenodata variable as part of the DESeq2 experimental design formula.
  • Size factors are estimated using a zero-tolerant variant of the geometric mean (see the phyloseq GitHub repository for details).
  • VST is applied as described in the phyloseq GitHub repository.
  • Data subjected to VST are separately saved as a phyloseq object (data converted to the DESeq2 format are not VST-transformed).

Output

The tool produces the following output files:

  • phyloseq object with transformed OTU counts (ps_xxxx.Rda; possible file names are ps_clr.Rda, ps_relabund.Rda, ps_hellinger.Rda and ps_vst.Rda)
  • OTU data converted to DESeq2 format (deseq2.Rda)
The data set converted to DESeq2 format is intended for use with the "Differential OTU abundance analysis using DESeq2" tool, while the other output files can be used for alternative analyses (e.g. PERMANOVA).
