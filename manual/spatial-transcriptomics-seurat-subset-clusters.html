Seurat v4 -Subset out anatomical regions based on clusters

Description

This tool filters out anatomical regions based on clusters to focus on a subset of data. After subsetting the data, the data must be renormalized using SCTransform and PCA must be run again. The subsetted object is then visualised using SpatialDimPlot.

Parameters

  • Subset of clusters [1,2,3,4,5]
  • Number of variable genes to return in SCTransform [3000]
  • Number of PCs to compute in PCA [50]

Details

Provide the Seurat R-object (Robj) as input.

For more details, please check the Seurat tutorials.

Output

  • seurat_obj.Robj: This is the Seurat R-object to pass to the next Seurat tool, or to import into R. However, this is not viewable in Chipster.
