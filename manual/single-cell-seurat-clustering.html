Seurat -Clustering

Description

This tool clusters cells and visualizes the results using t-SNE and UMAP plots.

Parameters

  • Number of principal components (PCs) to use [10]
  • Resolution for granularity [0.5]
  • Perplexity, expected number of neighbors in t-SNE plot [30]
  • Point size in t-SNE and UMAP plots [30]
  • Provide a list of average expressions in each cluster [no]

Details

Cells are clustered using principal components (PCs) rather than genes. You need to input the Seurat R-object that was generated using the Seurat PCA -tool. That tool also produced PC heatmaps and an elbow plot, which help you to decide the number of PCs to use.

If you have a low number of cells in your data, try lowering the perplexity parameter (the expected number of neighbours).

Graph-based clustering is performed using the Seurat function FindClusters, which first constructs a KNN graph using the Euclidean distance in PCA space, and then refines the edge weights between any two cells based on the shared overlap in their local neighborhoods (Jaccard's distance). It then divides the graph into clusters using the Louvain algorithm which optimizes the standard modularity function (please see the links below for more information).

The resolution parameter sets the 'granularity', where increased values leads to a higher number of clusters. It has been found that this parameter, when set between 0.6-1.2, typically returns interpretable results for single cell datasets of around 3k cells. Optimal resolution often increases for larger datasets. Use a value above 1.0 if you would like to obtain a larger number of communities, and below 1.0 if you would like a smaller number of communities.

To visualize the clusters, non-linear dimensional reduction is performed using t-SNE and UMAP on the same PCs as used for the graph-based clustering. The plots in clusterPlots.pdf are then colored by the clustering results. Cells belonging to the same cluster should co-localize on the t-SNE plot because t-SNE aims to place cells which have a similar local neighborhood in a high-dimensional space together, yet in a low-dimensional space. The perplexity parameter is an estimate about the number of close neighbors each cell has, so it allows you to balance attention between the local and global aspects of the data (read more). If you have a low number of cells, try lowering the perplexity value. Unlike t-SNE, UMAP also considers the global aspects of the data. Downstream, you can choose whether you would like to plot t-SNE or UMAP plots.

The tool can also return a table with expression for an 'average' single cell in each cluster. Read more about the AverageExpression function.

For more details, please refer to the Seurat tutorials
The Seurat clustering approach was predominantly inspired by the manuscripts SNN-Cliq, Xu and Su, Bioinformatics, 2015 and PhenoGraph, Levine et al., Cell, 2015 which applied graph-based clustering approaches to scRNA-seq data and CyTOF data, respectively.


Output

  • seurat_obj_clustering.Robj: The Seurat R-object to pass to the next Seurat tool, or to import into R, but not viewable in Chipster.
  • clusterPlot.pdf: Cluster visualization in t-SNE and UMAP plots
  • aver_expr_in_clusters.tsv: A table for "average expression" for each gene in each cluster
