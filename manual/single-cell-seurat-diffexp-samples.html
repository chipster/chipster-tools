Seurat -Find conserved cluster markers and differentially expressed genes (DEGs) in multiple samples

Description

For a given cluster, this tool provides the following information:

  • cell type markers conserved across all conditions (conserved_markers.tsv)
  • differentially expressed genes (DEGs) between conditions (de-list.tsv)
OR, in case of more than two samples:

  • DEGs of each sample compared to all other samples (de-list_samplenameVsAllOthers.tsv)
Both conditions and samples are determined using the Setup tool when using the Sample or Group name parameter.

Parameters

  • Name of the cluster [3]
  • Return only positive marker genes [TRUE]
  • Fold change threshold for conserved markers in log2 scale [0.25]
  • Fold change threshold for DEGs in log2 scale [0.25]
  • p-value cutoff for conserved markers [0.05]
  • Adjusted p-value cutoff for DEGs [0.05]
  • Minimum number of cells in one of the groups for conserved markers [3]

Details

Please provide the combined Seurat object.

Select the cluster you would like to inspect by setting its name in the parameter field. As an example, the parameter is set to cluster "3".

You can filter both the conserved marker genes and the DEGs using the parameters. By default, only genes with adjusted p-values of less than 0.05 are listed in the results table. Additionally, only positive markers are kept. Please note that Seurat adjusts p-values using the Bonferroni method for multiple testing correction.

In the analysis, you can opt to include only genes whose average log fold change is higher than a selected value. This prefiltering speeds up the analysis and narrows down the result list.

The result table de-list.tsv contains genes which are differentially expressed (DE) between the two conditions in the selected cluster. In case of more than 2 samples, there will be as many files as there are samples, named de-list_samplenamevsAllOthers.tsv. The columns in these files include:

  • p_val = p-values for the DEGs (if the p-value is larger, there is a greater likelihood that the gene is in the list just by chance)
  • p_val_adj = adjusted p-value. This value is corrected using multiple testing correction: when we test thousands of genes, we may identify some DEGs just by chance. There are different methods to correct for this, and Seurat specifically uses the Bonferroni method. When filtering the table and reporting your results, please use these values.
  • avg_logFC = average log2 fold change (how much higher or lower the expression of this gene is in the chosen cluster between conditions)
  • pct.1 = the percentage of the cells in condition 1 show some expression for this gene. Please note: numbering is based on alphabetical order: earlier in the alphabets -> 1, the other one -> 2. So for example, "CTRL" -> 1 and "STIM" -> 2, since C is before S in the English alphabet.
  • pct.2 = the percentage of cells in condition 2 show some expression for this gene

conserved_markers.tsv contains a ranked list of conserved marker genes for the chosen cluster (= genes that show specific expression in this particular cell type or cluster, regardless of the treatment). This file has as many column groups as number of samples in the data. In addition to the columns listed above, this table contains the following columns:

  • max_pval = maximum unadjusted p-value of the samples
  • min_pval = minimum unadjusted p-value of the samples

For more details, please refer to Seurat tutorials for multiple sample analysis.

Output

  • conserved_markers.tsv: Conserved cell type markers across conditions
  • de-list.tsv: DEGs between conditions
OR
  • several de-list_samplenameVsAllOthers.tsv files: DEGs between each sample and all other samples
