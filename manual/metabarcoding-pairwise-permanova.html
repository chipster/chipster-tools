Post-hoc pairwise PERMANOVA for OTU abundance data

Description

Performs post-hoc pairwise permutational analyses of variance (PERMANOVA, 999 iterations), using distance matrices derived from OTU abundance data. Accepts a single phenodata variable and applies a Benjamini-Hochberg correction to account for multiple testing. This tool should only be used after obtaining a statistically significant (p < 0.05) result using the "Global PERMANOVA for OTU abundance data" tool.

Parameters

  • Phenodata variable used for pairwise comparisons

Details

The tool requires the following input files:

  • An Rda file including phyloseq object sample data and a distance matrix (ps_dist.Rda, produced using the tool "Distance matrices and ordinations")
  • Data groupings are automatically fetched from a phenodata file linked to a Mothur shared file (file.opti_mcc.shared)
The tool employs the following R functions and packages:
  • pairwise.perm.manova (RVAideMemoire)

Output

The tool produces the following output files:

  • Text summary of pairwise PERMANOVA output (pairwise_permanova.txt)
