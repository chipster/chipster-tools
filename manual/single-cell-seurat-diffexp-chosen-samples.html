Seurat -Find DE genes between chosen samples

Description

For a given cluster, this tool gives you the differentially expressed genes between the samples/conditions listed by the user. Conditions/samples are determined in the Setup tool with the Sample or Group name parameter. If you are listing multiple samples, separate them with comma (,).

Parameters

  • Name of the samples to compare to
  • Name of the samples to compare with
  • Name of the cluster [3]
  • Return only positive marker genes [TRUE]
  • Fold change threshold for differentially expressed genes in log scale [0.25]
  • Adjusted p-value cutoff for differentially expressed genes [0.05]

Details

As inputs, give the combined Seurat object.

Select the cluster you want to inspect by setting its name in the parameter field. As an example, the parameter is set to cluster "3".

Select the samples or list of samples which you want to compare to each other by typing the sample names to the parameter fields. If you are listing multiple samples, use comma as separator: for example, type: "CTRL, CTRL_2" (withouth the hyphens "") in first field and "STIM, STIM_2" in the second.

You can filter the differentially expressed genes using the parameters. By default, only genes with adjusted p-value < 0.05 are listed in the result table and only positive markers are kept. Note that Seurat adjusts p-values using the rather conserved Bonferroni method for multiple testing correction.

You can choose to include in the analysis only genes whose average fold change is higher than a selected value. This prefiltering speeds up the analysis and narrows down the result list.

The result table contains the genes which are differentially expressed between the conditions in the selected cluster. The columns include

  • p_val = p-values for the differentially expressed genes (larger the p-value -> higher the likelihood that the gene is in the list just be chance)
  • p_val_adj = adjusted p-value. This value is corrected for multiple testing: when we test thousands of genes, we can get some statistically significantly differentially expressed genes just by chance. There are different methods to correct for this, Seurat uses the Bonferroni method. When filtering the table and reporting your results, use this value.
  • avg_logFC = average log2 fold change (how much higher or lower the expression of this gene is in the chosen cluster between the conditions
  • pct.1 = what percentage of the cells in the condition 1 show some expression for this gene. Note, that numbering is based on alphabets: earlier in the alphabets -> 1, the other one -> 2. So for example, "CTRL" -> 1 and "STIM" -> 2, since C is before S in alphabets.
  • pct.2 = what percentage of the cells in the condition 2 show some expression for this gene

For more details, please check the Seurat tutorials for multiple sample analysis.

Output

  • de-list_samplenames_vs_Samplesnames_in_clusterclustername.tsv files: Differentially expressed genes between the user defined samples
