Seurat -Find differentially expressed genes (DEGs) between chosen samples

Description

For a given cluster, this tool provides a list of the differentially expressed genes (DEGs) based on the conditions of the samples. Conditions and samples are selected in the Setup tool, using the Sample or Group name parameter. If you are listing multiple samples, separate them using comma(s) (,).

Parameters

  • Name of the samples to compare to
  • Name of the samples to compare with
  • Name of the cluster [3]
  • Return only positive marker genes [TRUE]
  • Fold change threshold for differentially expressed genes (DEGs) in log scale [0.25]
  • Adjusted p-value cutoff for differentially expressed genes [0.05]

Details

Please input the combined Seurat object.

Select the cluster you would like to inspect by setting its name in the parameter field. As an example, the parameter is set to cluster "3".

Select the sample(s) which you would like to compare with one another by typing the sample names in the parameter fields. If you are listing multiple samples, use a comma (,) as a separator: for example, type: CTRL, CTRL_2 in first field and STIM, STIM_2 in the second field.

You can filter the DEGs using the parameters. By default, only genes with adjusted p-values of less than 0.05 are listed in the results table and only positive markers are kept. Please note that Seurat adjusts p-values using the Bonferroni method for multiple testing correction.

Alternatively, you can choose to include only genes whose average fold change is higher than a selected value. This prefiltering accelerates the analysis and generates fewer DEGs in the results list.

The results table contains DEGs in the selected cluster. The columns include:

  • p_val = p-values for DEGs (larger the p-value -> higher the likelihood that the gene is in the list just by chance)
  • p_val_adj = adjusted p-value. This value is generated after applying multiple testing correction. When thousands of genes are tested, some DEGs are generated by chance. While different methods exist to correct for this, Seurat uses the Bonferroni method. When filtering the table and reporting results, please use this value.
  • avg_logFC = average log2 fold change (how much higher or lower the expression of this gene is in the chosen cluster between conditions)
  • pct.1 = what percentage of the cells in condition 1 show some expression for this gene. Please note, numbering is based on alphabetical order: earlier in the alphabets -> 1, the other one -> 2. For example, CTRL -> 1 and STIM -> 2, as the letter C comes before S when alphabetized.
  • pct.2 = what percentage of the cells in condition 2 show some expression for this gene

For more details, please refer to the Seurat tutorials for multiple sample analysis.

Output

  • de-list_samplenames_vs_Samplesnames_in_clusterclustername.tsv files: DEGs between the user defined samples
