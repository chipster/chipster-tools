Seurat -Visualize genes with cell type specific responses in multiple samples

Description

This tool gives you plots showing user defined markers/genes across the conditions. This tool can be used for Seurat objects containing two or more samples.

Parameters

  • Normalisation method used previously [LogNormalize]
  • Markers to plot [CD3D, CREM, HSPH1, SELL, GIMAP5]
  • Scale data in split dot plot [no]
  • Visualisation with tSNE, UMAP or PCA in feature plot [UMAP]
  • Plotting order of cells based on expression in feature plot [no]
  • Determine color scale based on all features in feature plot [no]

Details

As input, give a combined Seurat object.

Select the markers you want to inspect by listing them in the parameter row. The markers can also be given in then form of a text file to the input field. Separate the names with comma (,). As an example, the parameter is set to markers "CD3D, CREM, HSPH1, SELL, GIMAP5".

The "split dot plot" can be useful for viewing conserved cell type markers across conditions, showing both the expression level and the percentage of cells in a cluster expressing any given gene. The size of the dot encodes the percentage of cells within a class (=cluster), while the color encodes the AverageExpression level of 'expressing' cells (stronger color, more experssion). By default, the raw expression data is used but the user can choose to use scaled data by setting the parameter value for "plotting order of cells based on expression in feature plot" to "yes".

Another useful way to visualize changes in gene expression is with "feature plots" that are also visualized by splitting them by conditions. By default, the color scale in the feature plots are determined for each gene individually and may differ between genes. If you wish to compare gene expression between different genes, it is useful to set the parameter "determine color scale based on all features in feature plot" to "yes" so that the color scale is the same for all of the visualized genes.

In case of more than 2 samples, the plots are divided to multiple pages.

For more details, please check the Seurat tutorials for multiple sample analysis.

Output

  • split_dot_plot.pdf: Plots showing user defined markers/genes across the conditions
