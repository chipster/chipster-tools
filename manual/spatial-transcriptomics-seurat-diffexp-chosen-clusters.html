Seurat v4 -Identify spatially variable genes based on clusters

Description

Find differentially expressed genes (DEGs) between two clusters.

Parameters

  • Cluster to identify markers [1]
  • Cluster for comparison [2]
  • Test used for differential expression [wilcox]

Details

As an input, you can provide the Seurat object generated using the tool: Seurat v4- PCA, clustering and visualisation.

Select the clusters in which you would like to identify spatially variable genes. As an example, differential expression is performed between clusters 1 and 2. The first cluster is used as the identity class to define markers and the second cluster is used as the identity class for comparison.

Select the test used for differential expression. Options for this are the Wilcoxon Rank Sum Test (wilcox) or the hurdle model tailored to scRNA-seq analysis called MAST. Wilcox is set as the default.

The FindMarkers() function finds DEGs for cell type identity classes (e.g., CD4 T cells, T Helper cells). This function has a parameter for min.pct, or genes found in a minimum percentage of spots in the two clusters, where the default is set to 0.1 or 10%. The function also includes the logfc.threshold parameter, which is a threshold where genes need to be at least a certain X-fold difference between the two groups of cells; the default is set to 0.25 or 25%.

Output
