Annotate cells with custom SummarizedExperiment reference using SingleR

Description

Annotate clusters and cells of a seurat object with custom reference.

Parameters

  • Pruning [yes]
  • Fine tuning [yes]
  • Label size[4]
  • Width [10]
  • Height [10]

Details

SingleR leverages reference transcriptomic datasets of pure cell types to infer the cell of origin of each of the single cells independently. SingleR’s annotations combined with Seurat, a processing and analysis package designed for scRNA-seq, provide a powerful tool for the investigation of scRNA-seq data.

For more details about SingleR, please check:
The SingleR GitHub repository And The SingleR Book

We would like to thank Iivari Kleino for his kind assistance with the development of this tool.

Output

  • seurat_obj_annotated.Robj: Seurat object that has cell type annotations per cell and per cluster. The Seurat R-object to pass to the next Seurat tool, or to import to R. You cannot open this file in Chipster, but you can view the contents of the object using the Extract information from Seurat object tool.
  • Plots.pdf: A UMAP showing celltype annotations on clusters and cells. Additionally, if pruning is TRUE, also plotScoreHeatmap and DeltaDistribution plots are plotted.
