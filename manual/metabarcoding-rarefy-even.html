Rarefy to even depth

Description

Resamples a phyloseq OTU table such that all samples have the same sequencing depth, and saves the resulting object as an Rda file.

Parameters

None

Details

The tool requires the following input files:

  • A phyloseq object in Rda format
  • Data groupings are automatically fetched from a phenodata file linked to a Mothur shared file (file.opti_mcc.shared)
The tool employs the following R functions and packages:
  • rarefy_even_depth() (phyloseq)

Output

The tool produces the following output files:

  • Phyloseq object with resampled OTU table (ps_rarefied.Rda)
  • Text summary of phyloseq object following rarefying to even depth (ps_rarefied.txt)
