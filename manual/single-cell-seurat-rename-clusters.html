Seurat -Rename clusters

Description

This tool allows you to rename clusters.

Parameters

  • Point size in t-SNE and UMAP plots [1]

Details

You can use this tool to rename clusters. For example, instead of "0", "1", or "2", you can provide cluster names according to recognised cell markers or cell types, such as "NK", "CD8+ T", or "Platelet". To do so, you will need to create a tab separated file (.tsv) using a program like Excel or another text editor. The file should contain old cluster names in first column and the new names in the second column.

The layout should look similar to this:
Cluster ID Cluster name
0 Naive CD4+ T
1 CD14+ Mono
2 Memory CD4+
3 B

Please note that you will need to provide the old cluster names (0, 1, 2...) in the same order as in the Seurat R-object. You will also need to provide the name of each cluster (you cannot have empty cells in the input table).

For more details, please refer to the Seurat tutorials.

Output

  • seurat_obj_renamed.Robj: The Seurat R-object to pass to the next Seurat tool, or to import into R, but it is not viewable in Chipster.
  • clusterPlotRenamed.pdf: t-SNE and UMAP plots are used to visualize clusters.
