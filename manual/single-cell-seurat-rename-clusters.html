Seurat -Rename clusters

Description

With this tool, you can rename the clusters.

Parameters

  • Point size in tSNE and UMAP plots [1]

Details

With this tool, you can rename the clusters: for example, instead of "0", "1", "2"... give clusters names according the recognised cell markers or cell types, such as "NK", "CD8+ T", "Platelet". For this purpose, you need to create (for example with in Excel, text editor...) a tab separated .tsv file, which has the old cluster names in first column, and the new names in the second column.

Like this:
Cluster ID Cluster name
0 Naive CD4+ T
1 CD14+ Mono
2 Memory CD4+
3 B

Note, that you need to give the same old cluster names (0, 1, 2...) in same order as they are in the Seurat R-object. You need to give name to each cluster (you can't have empty cells in the input table).

For more details, please check:
The Seurat tutorials

Output

  • seurat_obj_renamed.Robj: The Seurat R-object to pass to the next Seurat tool, or to import to R. Not viewable in Chipster.
  • clusterPlotRenamed.pdf: Cluster visualization in tSNE and UMAP plots, heatmap showing the expression of top ten marker genes (in terms of fold change) for each cluster.
