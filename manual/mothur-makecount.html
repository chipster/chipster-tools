Combine FASTQ files into one FASTA file and make a Mothur count file

Description

Combines FASTQ files of each sample to one FASTA file and creates a Mothur count file. Input should be a tar package containing FASTQ files. Input files can be gzipped.

Parameters

None

Details

For the Mothur-based tools in Chipster your samples needs to be in one FASTA file. In order to assign sequences to a specific sample, you need also a Mothur count file. If you have single end data (e.g. Ion Torrent) and your samples are in separate FASTQ files or if you have paired end reads which you have merged into contigs with VSEARCH, you can convert the FASTQ files to FASTA and combine them with this tool which generates also the count file.

This tool is based on the Mothur tools make.count, merge.files and summary.seqs.

Output

The output consists of the following files:

  • sequences.fasta.gz: Compressed FASTA file containing sequences from all the samples
  • sequences.count_table: Mothur count file indicating what sequence belongs to what sample and how many times
  • sequences-summary.tsv: Summary statistics for the sequences
