Sequence numbers, rarefaction curve and alpha diversity estimates for phyloseq OTU data

Description

Lists per-sample sequence numbers, plots rarefaction curves and tabulates and creates a boxplot on alpha diversity estimates (observed no. of OTUs, Chao1 and Shannon indices, Pielou's evenness) for OTU abundance data stored as a phyloseq object and calculates means of alpha diversity estimates as well as Wilcoxon rank sum test if the phenodata variable is selected.

Parameters

  • Phenodata variable tabulated with alpha diversity estimates and to calculate means of alpha diversity estimates as well as Wilcoxon rank sum test
  • Name of the first sample group from the Phenodata variable for Wilcoxon rank sum test
  • Name of the second sample group from the Phenodata variable for Wilcoxon rank sum test

Details

The tool requires the following input files:

  • A phyloseq object in Rda format
  • Data groupings are automatically fetched from a phenodata file linked to a Mothur shared file (file.opti_mcc.shared)
The tool employs the following R functions and packages:
  • Rarefaction curve: rarecurve() (vegan)
  • Alpha diversity indices: estimate_richness() (phyloseq) and evenness() (microbiome)
  • Wilcoxon rank sum test
Note that the alpha diversity estimates are only reliable when using raw OTU data, as many diversity metrics depend on the presence of singletons and doubletons in the data set under analysis.

If the phenodata variable is selected, this tool calculates also the means, standard error and standard deviation of alpha diversity estimates.
If there is two groups, this tool calculates also Wilcoxon rank sum test between the two groups. If there is more than two groups, please select which two groups, you would like to compare.
This tool will also create a boxplot visualizing the alpha diversity estimates if the phenodata variable for showing grouping is selected. In addition, the Wilcoxon rank sum test results are plotted to the the boxplot.

Output

The tool produces the following output files:

  • Rarefaction curve (ps_rarecurve.pdf)
  • Text summary of alpha diversity indices (ps_imported.txt)
  • Alpha diversity estimates visualized in a boxplot (plot_richness.pdf)
