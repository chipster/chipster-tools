SingleR cluster annotation

Description

This tool can be used to annotate clustered scRNA-seq data by comparing it to the CellDex annotation datasets.

Parameters

  • CellDex reference to use [Monaco Immune data]
  • Point size in plot [1]

Details

SingleR is an automatic annotation method for single-cell RNA sequencing (scRNA-seq) data (Aran et al., 2019). Given a reference dataset of samples (scRNA-seq or bulk) with known labels, it labels new cells from a test dataset based on similarity to the reference. The CellDex package provides access to several reference datasets (mostly derived from bulk RNA-seq or microarray data) through dedicated retrieval functions. You can select the CellDex package to be used as the reference. Please input the Seurat object including clustering information.

For more details, please refer to the following:

  • SingleR package documentation
  • CellDex package documentation
  • Using SingleR to annotate single-cell RNA-seq data vignette

Output

  • seurat_obj_singler_annotations.Robj: The Seurat R-object including the annotations in the metadata field. While it is not viewable in Chipster, it can be exported and examined in R.
  • singleR_annotation_plots.pdf: Plots showing annotation diagnostics
  • annotations_main.tsv: Main level annotations table
  • annotations_fine.tsv: Fine level annotations table
