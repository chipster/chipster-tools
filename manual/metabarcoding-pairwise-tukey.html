Post-hoc pairwise Tukey's HSD for OTU abundance data

Description

Performs post-hoc pairwise comparisons of group dispersions using a Tukey's Honestly Significant Difference (HSD) test. Should only be used after a significant (p < 0.05) PERMDISP result. Performs up to three tests (for three different phenodata variables), depending on preceding steps taken when using the PERMDISP tool.

Parameters

  • Number of PERMDISP analyses performed during the preceding (PERMDISP) analysis step (1-3)

Details

The tool requires the following input files:

  • An Rda file including dispersion and test data (ps_disp.Rda, produced using the tool "PERMDISP for OTU abundance data")
The test is completed using the TukeyHSD() function available via the base R package stats.

Output

The tool produces the following output files:

  • Text summary of pairwise Tukey's HSD output (pairwise_tukey_table.txt)
  • Plot(s) displaying pairwise Tukey's HSD comparisons (pairwise_tukey_plot.pdf)
