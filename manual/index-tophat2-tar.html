Create TopHat2 index

Description

This tool creates an index Tar package for the TopHat2 aligner.

Parameters

None

Input

  • input.fa: reference genome fasta file
  • input.gtf: optional reference genome gtf file

TopHat2 uses Bowtie2 index, which is created from the genome FASTA file. The original FASTA file is included into the index Tar package, so that TopHat2 doesn't have to reconstruct it from the index in the alignment tool.

Supplying a GTF file containing known gene and exon locations is recommended, because it improves the alignment process. In this case a separate transcriptome index for TopHat2 is created and included in the package too.

Output

This tool produces Tar package that can be given to the tools "TopHat2 for single end reads and own genome" and "TopHat2 for paired end reads and own genome".

Reference

This tool is based on the TopHat package. Please cite the following article:

Kim D, Petrtea G, Trapnell C, et al. TopHat2: accurate alignments of transcriptomes in the presence of insertions, deletions and gene fusions. Genome Biology (2013) 14: R36.
