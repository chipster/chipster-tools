Seurat -Remove clusters

Description

With this tool you can remove clusters from your data. You can either list the clusters you want to remove, or the clusters you wish to include.

Parameters

  • Do you wish to remove or include these clusters [remove]
  • Names of the clusters to remove or include [3]

Details

As input, give the Seurat R-object (Robj) after clustering step.

If you give names of clusters not present in the data, the tool will give you an error.

For more details, please check the Seurat tutorials.

Output

  • seurat_obj.Robj: The Seurat R-object to pass to the next Seurat tool, or to import to R. You cannot open this file in Chipster, but you can view the contents of the object using the Extract information from Seurat object tool.
  • ClusterPlots.pdf: Cluster plots where you can confirm that correct clusters were removed/included.
