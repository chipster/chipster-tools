Seurat -Visualize genes

Description

With this tool, you can visualize selected genes using violin, UMAP, or ridge plots. You can input the Seurat R-object (.Robj) and gene of interest name(s) (e.g., MS4A1, LYZ, PF4...). The gene name(s) can also be provided in a text file in the input field.
Please note: If you list multiple gene names, use commas (,) as separators.

Parameters

  • Gene name
  • Point size in cluster plot [1]
  • Add labels on top of clusters in plot (yes, no) [no]
  • Visualisation with t-SNE, UMAP or PCA [UMAP]
  • Plotting order of cells based on expression (yes, no) [no]
  • Provide a list of average expression and percentage of cells expressing the selected gene in each cluster (yes, no) [no]

Details

Both the Violin plot and ridge plot show the probability density at different expression levels of the gene for each cluster. The color coding of the clusters helps with visualizing the expression of marker gene(s). You can look for marker genes specific to each cluster.

The UMAP plot is useful for visualizing the expression of the selected marker gene in the context of all the cells. It also helps validate the specificity of the marker or the quality of the clustering. It can be useful to plot cells in order of expression, particularly if cells expressing a given gene are are difficult to detect.

You can generate two tables, one which shows the average expression for each gene and one that shows the percentage of cells expressing the gene(s) in each cluster.

For more details, please refer to the Seurat tutorials.
