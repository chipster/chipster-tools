Convert FASTQ to FASTA

Description

Given a FASTQ file, this tool converts it to FASTA format. Reads can be also filtered for unknown nucleotides (Ns), and sequence identifiers can be renamed as numbers.

Parameters

  • Remove sequences with unknown nucleotides (yes, no) [yes]
  • Rename sequence identifiers as numbers (yes, no) [no]
  • Write a log file (yes, no) [no]
  • Base quality encoding used (Sanger, Illumina GA v1.3-1.5) [Sanger]

Details

Converts FASTQ files to FASTA format. Note that the base quality information available in FASTQ format is not available in FASTA. If you decide to remove sequences which contain unknown nucleotides, you might like to get a log file indicating how many sequences were discarded.

Output

FASTA file containing the reads.

Reference

This tool is based on the FASTQ-to-FASTA converter of the FASTX package.
