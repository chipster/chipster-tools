Visualisation / Volcano plot

Description

Creates a volcano plot from results of a statistical test.

Parameters

  • P-value column
  • Fold change column
  • Gene names (0...50) [10]
  • Gene labels
  • Image width (200...3200) [600]
  • Image height (200...3200) [600]

Details

This tool creates a volcano plot from data containing genewise p-values and fold changes. In the plot, red dots represent genes below the p-value threshold. The threshold is indicated with a dashed horizontal line.
