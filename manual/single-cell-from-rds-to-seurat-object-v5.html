Seurat v5 -From RDS file to Seurat object

Description

This tool converts an RDS file into a Seurat object.

The RDS file given as input must contain Seurat object information created with Seurat version 5 to be converted into a Seurat object.

This tool is useful in situations where single-cell data has been previously analysed with Seurat v4 and then saved as an RDS file. By converting the RDS file back into a Seurat object with this tool, the analysis can be continued in Chipster.

Details

After converting an RDS file into a Seurat object, you can use the "Seurat v5 -Extract information from Seurat object" tool to view the contents of the Seurat object in Chipster.

Output

  • seurat_obj.Robj: The Seurat R-object to pass to the next Seurat tool.
