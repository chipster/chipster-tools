Trim primers and barcodes and filter reads

Description

Given a FASTA or FASTQ file and oligos file, this tool trims primers and barcodes and filters reads for several criteria, including uniqueness. You can give a QUAL file as well.

Parameters

  • Use reverse complement (yes, no) [no]
  • Minimum average quality of sequence (0-40) []
  • Minimum average quality of window (Integer) []
  • Window size (Integer) []
  • Window step size (Integer) []
  • Maximum ambiguous bases (0-10) []
  • Maximum homopolymer length (0-50) []
  • Minimum sequence length (0-1000) []
  • Maximum sequence length (0-1000) []
  • Maximum differences to primer sequences (0-10) []
  • Maximum differences to barcode sequences (0-10) []

Details

This tool trims from the sequences the primers and barcodes provided in the oligos file, and filters out reads which don't fulfill the user-defined criteria. Also duplicate reads are removed and a count_table is produced showing how many times each representative sequence occurs in the different samples.

The oligos file contains sample names for each barcode. Please note that Mothur does not accept hyphens in sample names, so you need to remove them from the oligos file if you have any.

This tool is based on the Trim.seqs, Unique.seqs and summary.seqs commands of the Mothur package.

Output

The analysis output consists of the following files:

  • trim.unique.fasta: Trimmed and filtered unique sequences.
  • trim.unique.count_table: Table showing how many times each representative sequence occurs in the different samples.
  • summary.trim.unique.tsv: Summary statistics for the sequences
