Build a celltype reference from a pre-annotated Seurat object

Description

With this tool you can build a SummarizedExperiment celltype annotation reference object by using a pre-annotated Seurat object.

Parameters

  • Aggregate cells [no]

Details

The SummarizedExperiment Bioconductor page

We would like to thank Iivari Kleino for his kind assistance with the development of this tool.

Output

  • SummarizedExperiment_reference.Robj: The reference object made by SummarizedExperiment by using pre-annotated Seurat Object. This is used with SingleR to annotate cells on an unannotated Seurat object.
