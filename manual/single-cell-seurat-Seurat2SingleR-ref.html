Build a celltype reference from a pre-annotated Seurat and transfer labels to unannotated Seurat

Description

With this tool you can build a SingleR celltype annotation reference object by using a pre-annotated Seurat object. You can also annotate cells on an unannotated seurat object.

Parameters

  • Aggregate cells [FALSE]
  • Pruning [TRUE]
  • Fine tuning [TRUE]

Output

  • Built-SingleR-ref.Robj: The reference object made by SingleR by using pre-annotated Seurat Object.
  • seurat_obj_annotated.Robj:(Only if unannotated Seurat object is inputted) The Seurat R-object to pass to the next Seurat tool, or to import to R. You cannot open this file in Chipster, but you can view the contents of the object using the Extract information from Seurat object tool.
  • Plots.pdf: A UMAP showing celltype annotations on clusters and cells. Additionally, if pruning is TRUE, also plotScoreHeatmap and DeltaDistribution plots are plotted.
