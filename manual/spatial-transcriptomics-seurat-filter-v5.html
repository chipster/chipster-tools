Seurat v5 -Filter spots

Description

This tool filters out bad quality spots based on mitochondrial, ribosomal and hemoglobin transcript percentage.

Parameters

  • Filter out spots which have higher mitochondrial transcript percentage [20]
  • Filter out spots which have lower ribosomal transcript percentage [0]
  • Filter out spots which have higher hemoglobin transcript percentage [20]

Details

The tool filters out spots with damaged cells based on mitochondrial, ribosomal and hemoglobin transcript percentage.

As input, you can give the Seurat object from the tool Seurat v5 -Setup and QC. You can also use the QC-plots.pdf from the Seurat v5 -Setup and QC tool to estimate the parameters for this filtering step.

In multiple sample analysis, the filtered Seurat objects can be combined after this step using the tool Seurat v5 -Combine multiple samples, normalization, PCA, clustering and visualization".

Output

  • seurat_obj_filtered.Robj: The Seurat R-object to pass to the next Seurat tool, or to import into R. You cannot open this file in Chipster but you can view the contents of the object using the Seurat v5 -Extract information from Seurat object tool.
