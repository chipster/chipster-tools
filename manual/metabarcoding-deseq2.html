Differential OTU abundance analysis using DESeq2

Description

Tests for OTU differential abundances using DESeq2. A Benjamini-Hochberg multiple inference correction is applied to the data, with OTUs filtered using a cut-off value of 0.01. While the analysis can be performed using a phenodata variable comprising >2 sample groups, output files are restricted to comparisons between two groups. Produces a result table and plots OTU differential abundances at a user-specified level of biological organization. Note that the input file should not contain transformed OTU abundance data (raw counts are required, see Details for information on the input file).

Parameters

  • Number of sample groups under the phenodata variable used for the analysis (two or above two; the default is two)
  • Lower-level taxonomic classification used for differential abundance plotting (Class, Order, Family, Genus; the default is Class)
  • Higher-level taxonomic classification used for differential abundance plotting (Phylum, Class, Order, Family; the default is Phylum)
  • Groups 1 and 2 for DESeq2 contrasts (if there are >2 sample groups under the phenodata variable used for the analysis)

The lower-level taxonomic classification should correspond to a level of biological organization that is lower than the higher-level taxonomic grouping. OTUs classified as NA at the lower taxonomic level are filtered out to enable clearer plots.

Details

The tool requires the following input files:

  • A phyloseq object converted into the DESeq2 format (i.e. the file "deseq2.Rda" generated by the "Transform OTU counts" tool). This file should contain non-transformed data (raw counts) only.
The tool employs the following R functions and packages:
  • DESeq() and results() (DESeq2)
  • ggplot() (ggplot2)
Technical notes:
  • The DESeq() function employs a parametric fit type (see DESeq2 for details).

Output

The tool produces the following output files:

  • A DESeq2 results table in TSV format (deseq2_otutable.tsv)
  • Summary plot showing log2 fold changes (deseq2_otuplot.pdf)
