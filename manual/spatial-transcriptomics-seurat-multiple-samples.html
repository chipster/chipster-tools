Seurat v4 -Combine multiple samples

Description

This tool can be used to merge or integrate multiple Seurat objects for downstream joint analysis.

Parameters

  • Combining method [merge]

Details

As inputs, give the Seurat objects generated with the Seurat Setup tool and preprocessed with the SCTransform tool.

Merge vs integration as the combining method

The "combining method" parameter determines which method is used to combine. By default, merge is used, but the user can also choose to use integration. Merge simply combines the objects into one.

The integration method uses Canonical correlation analysis (CCA) to identify common sources of variation across datasets. After this, the tool identifies anchors, generates scores (=weights) for these anchors and then uses the anchors to integrate the datasets. The integration method should be used to remove strong batch effects present in the data.

Further information

For more details on integration, please check the vignette for introduction on integration with single-cell data "Introduction to scRNA-seq integration".

Output

  • seurat_obj.Robj: The Seurat R-object to pass to the next Seurat tool, or to import into R, but not viewable in Chipster.
