Seurat v5 -Visualize cell types after integration with scRNA-seq data

Description

This tool visualizes the results after Seurat v5 -Integration with scRNA-seq data. This tool plots the chosen genes with SpatialFeaturePlot. It also identifies and visualizes the top expressed spatially variable genes that are computed using the moransi or markvariogram method.

Parameters

  • Genes to plot ["L2/3 IT, L4"]
  • Method to use [moransi]
  • Numbers of features to plot [4]

Details

The spatially variable genes are computed similarly as in the tool Seurat v5 -Identify spatially variable genes without pre-annotation except with prediction scores as the “marks” rather than gene expression. These features are then visualized in the SpatialPlot.

As the input, please provide the Seurat object generated using the tool Seurat v5 -Integration with scRNA-seq data.

Output

  • integration_plot.pdf
