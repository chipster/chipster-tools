Seurat v5 -Visualize cell types after integration with scRNA-seq data

Description

This tool visualizes the results after Seurat v5 -Integration with scRNA-seq data. This tool visualizes the predicted compositions of selected cell types in each of the spatial spots. It also identifies and visualizes predicted cell types whose location is spatially restricted. These spatially variable cell types are computed using the same Moran's I or Markvariogram methods as used in the tool "Seurat v5 -Identify spatially variable genes without pre-annotation".

As the input, please provide the Seurat object generated using the tool Seurat v5 -Integration with scRNA-seq data.

Parameters

  • Cell types to plot ["L2/3 IT"]
  • Method to use [moransi]
  • Number of top spatially variable cell types to plot [4]
  • Determine color scale based on all cell types [no]
  • Multiple sample analysis [no]

Details

The user can select for which cell types the tool visualizes the predicted cell type compositions using the parameter "Cell types to plot". The spatially variable cell types are computed similarly as in the tool Seurat v5 -Identify spatially variable genes without pre-annotation except with prediction scores as the “marks” rather than gene expression values. You can choose between Moran's I (moransi) and Markvariogram (markvariogram) methods for computing these spatially variable cell types using the parameter "Method to use". You can also choose the number of spatially variable cell types visualized using the "Number of top spatially variable cell types to plot" parameter.

The user can also determine whether the color scale is based on all cell types or individual cell types. By default, the color scale is determined for each cell type individually and may differ between cells. If you wish to compare multiple cell types, it is useful to set this parameter to "yes" so that the color scale is the same for all cell types.

Note that if you performed multiple sample analysis i.e. if your samples have been merged or integrated into one single seurat object, select 'yes' as the value for parameter 'Multiple sample analysis'. However, this tool will still identify and visualize spatially variable cell types for each sample individually.

For more details on the spatial transcriptomics analysis pipeline, please check the Seurat tutorial.
