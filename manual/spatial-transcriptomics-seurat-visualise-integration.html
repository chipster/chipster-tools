Seurat v4 -Visualise integration

Description

This tool visualises the results of the "Seurat v4 -Integration with single-cell data". This tool plots the chosen features with SpatialFeaturePlot. It also identifies and visualises the top expressed spatially variable features computed using the markvariogram.

Details

The spatially variable features are computed similarly as in the tool "Seurat v4 -Identify spatially variable features using markvariogram" except with prediction scores as the “marks” rather than gene expression. These features are then visualised in the SpatialPlot.

As an input, give the Seurat object generated with "Seurat v4 -Integration with single-cell data".

Output

  • integration_plot.pdf
