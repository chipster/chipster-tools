Seurat -Visualize genes

Description

With this tool user can visualize selected genes with violin, feature (UMAP, tSNE or PCA) and ridge plot. As input the user gives the Seurat R-object (.Robj) and the name of the gene of interest (for example MS4A1, LYZ, PF4...). The gene name(s) can also be given in the form of a text file to the input field.
Note: If you list multiple gene names, use comma (,) as separator.

Parameters

  • Gene name(s) [MS4A1, LYZ]
  • Normalization method used previously [Global scaling normalization]
  • Feature plot visualization with tSNE, UMAP or PCA [UMAP]
  • Point size in feature plot [1]
  • Add labels on top of clusters in plot (yes, no) [no]
  • Plotting order of cells based on expression (yes, no) [no]
  • Determine color scale based on all features in feature plot (yes, no) [no]
  • For each gene, list the average expression and percentage of cells expressing it in each cluster(yes, no) [no]

Details

Violin plot and ridge plot show the probability density at different expression levels of the gene for each cluster. The color coding of the clusters helps to see in which clusters the marker gene is expressed. Good marker genes will show strongly in a single cluster.

The feature plot is useful for viewing the expression of the selected marker gene in the context of all the cells and helps validating the specificity of the marker or the quality of the clustering. It can be useful to plot the cells in the order based on expression, if the cells expressing given gene are getting buried.

You can request to get two tables which show for each gene the average expression and the percentage of cells expressing it in each cluster.

For more details, please check the Seurat tutorials.
