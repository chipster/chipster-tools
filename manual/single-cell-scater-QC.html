Scater - QC

Description

This tool gives as an output an .pdf file with several plots that can be used to estimate the quality of the data and compare samples. As an input, give a Seurat object. The plots vary a bit depending on what values are available in the Seurat input: if the cell-cycle scores are available (after the "Filter, normalize, regress and detect variable genes" -tool) those are used in plotting. This tool transforms the Seurat object into Scater format, and computes several QC metrics.

Parameters

None

Details

The Seurat object given as an input is converted into a Scater object (SCE). After this, several QC plots are printed in the output pdf file.

Plot 1: Most expressed features

Plot shows the features with the (50) highest average expression across all cells, along with their expression in each individual cell. In this plot, you can check which genes are the top 50 most expressed, and detect possibly overabundant genes. For example, you can see if there are mitochondrial genes (MT-) that you might want to remove from the analysis. Y-axis lists the top genes, and x-axis the percentage of total counts. Each small bar/tick represents a cell. The color of the tick represents the total number of features in that cell. The dot in each line shows the average epression.

Plot 2: Cumulative expression

Plot shows the relative proportion of the library size that is accounted for by the 1000 most highly expressed features for each cell. Y-axis is the cumulative proportion and x-axis number of the top expressers included. Each cell is presented as a line. This plot can be used to to look for differences in expression distributions between samples.

Plot 3: Comparing statistics

Several QC values plotted against each other. You can read in the Scater documentation for calculateQCmMtrics what each of the QC values stands for.
For example:
pct_counts_top_50_features: What percentage of the total counts is accounted for by the 50 highest-count features?
pct_counts_feature_controls: Percentage of the total counts counts for the cell that come from control features: mt.genes (mitochondrial genes, starting with "MT") are used as controls here.

Plots 4 and 5: Principal component comparisons

Here we see first 4 principal components plotted against each other. The percentage of the variability of the data explained by each PC is also reported. The dots representing the cells are colored based on mitochondrial % and total number of features.

Plots 6-9: Explanatory factors

We can investigate the relative importance of different explanatory factors with these plots. In the first two plots, a density plot is created for each variable, showing the distribution of R-squared across all genes. Scater calculates R-squared for a factor/variable by fitting a line for each gene against the factor/variable. R-squared is the proportion of variance for dependant variable explained by an independent variable. We can also identify PCs that correlate strongly to certain QC factor or meta-data values. The second two plots represent these. In both cases, some selected factors are plotted, and then the top 10 factors. If there are cell-cycle scores present in the Seurat object given as an input, the values for the phase, G2M score and S score are also plotted.

For more details, please check the Scater package in Bioconductor and the Scater article: McCarthy DJ, Campbell KR, Lun ATL, Willis QF (2017). “Scater: pre-processing, quality control, normalisation and visualisation of single-cell RNA-seq data in R.” Bioinformatics, 33, 1179-1186. DOI.

Output

  • QCplots.pdf: Quality control plots
