ChIP-seq / Remove duplicate genes

Description

This tool takes a list of gene identifiers and removes duplicates. The output file is compatible with downstream tools used for example for pathway analysis.

Parameters

  • ID column [ensembl_id]

Details

The identifier column can be selected from the parameter. If the file contains a column ensembl_id, it will be selected by default.

If the file has column location, then it's values affect which row among same IDs will be kept. The values are prioritised as follows:

  • upstream
  • overlapStart
  • inside
  • overlapEnd
  • downstream
  • any other value

Output

The analysis output is a table of annotated genes, which is compatible with downstream tools used for example for pathway analysis.

The file rows are sorted according to the text in the ID column.
