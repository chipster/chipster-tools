Seurat v4 -Visualise clusters

Description

With this tool user can visualise clusters separately on the tissue histology to distinguish the spatial localization of individual clusters. As input the user gives the Seurat R-object (.Robj) and the numbers of the clusters, for example 1,2,3,5.
Note: If you list multiple clusters, use comma (,) as separator.

Parameters

  • Clusters to plot [2,1]

Details

SpatialDimPlot shows the selected clusters on top of the tissue histology.

For more details, please check the Seurat tutorials.

Output

  • spatialdimplot.pdf
