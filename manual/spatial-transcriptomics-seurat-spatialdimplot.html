Seurat v4 -Visualise clusters

Description

With this tool, the user can visualise clusters separately on the tissue histology to distinguish the spatial localization of any individual clusters. The user can provide both the Seurat R-object (.Robj) and clusters to plot (e.g., 1,2,3,5) as inputs.
Please note: If you list multiple clusters, use commas (,) as separator(s).

Parameters

  • Clusters to plot [2,1]

Details

SpatialDimPlot shows the selected clusters on top of the histological tissue.

For more details, please check the Seurat tutorials.

Output

  • spatialdimplot.pdf
