Remove samples from a Phyloseq object

Description

This tool can be used to remove up to five user-specified samples from a phyloseq object. Produces a phyloseq object without the defined samples and saves it as an Rda file, and a summary of the subsetted phyloseq object.

Parameters

  • Samples to be removed (up to five)

Details

Names of samples to be removed (one to five) are typed into the provided text fields. The sample names should be written exactly as they appear in the phenodata table. If any of the entered sample names do not match those in the phenodata table, a warning is shown.

After removing samples, it is recommended to run the tool Remove OTUs with 0-20 ocurrences with the filtering threshold of 0. This step removes OTUs that occur only in the samples that were removed and have 0 occurrences in the current set of samples.

The tool requires the following input files:

  • A phyloseq object in Rda format
The tool employs the following R functions and packages:
  • subset_samples() (phyloseq)

Output

The tool produces the following output files:

  • Phyloseq object following removal of specified samples (ps_subset.Rda)
  • Text summary of the phyloseq object following sample removal (ps_subset.txt)
