Seurat -Extract cells in a cluster

Description

This tool subsets the data by only extracting cells belonging to a user-defined cluster.

Parameters

  • Name of the cluster [3]

Details

Please input the Seurat object created after one of the two clustering steps listed below:

  • Seurat v3 -Clustering and detection of cluster marker genes tool
  • Seurat v3 -Integrated analysis of two samples tool (used for two sample analysis)

Select the cluster you would like to inspect by setting its name in the parameter field. As an example, the parameter is set to cluster "3".

You can continue analysing this subsetted R-object by running the Seurat tools again; in the case you started with a single sample (subsetting after Seurat v3 -Clustering and detection of cluster marker genes tool), you can start by filtering and finding variable genes, and continue from there.
If you started with a combined Seurat object of two samples (subsetting after Seurat v3 -Integrated analysis of two samples tool), you can continue by running the same tool again.

Please note that the cell filtering (number of genes per cell, mito%) performed on the original dataset impacts this object as well.

Output

  • seurat_obj_subset.Robj: The Seurat R-object containing only the cells in the chosen cluster. This can be passed to the next Seurat tool, or imported into R, but is not viewable in Chipster.
  • QCplots.pdf: Quality control (QC) plots (similar to the ones in Setup tool)
