Seurat -Subset Seurat object based on gene expression

Description

This tool generates a subset of the data comprised of cells having expression in a user-defined gene. Please set the expression threshold parameter.

Parameters

  • Gene ["MS4A1"]
  • Expression level threshold [1]

Details

Please input the Seurat object.

Select a gene of interest that will be used to then subset the data (as an example, this parameter is set to "MS4A1"). Then select the expression value threshold (as an example, this parameter is set to 1).

Output

  • seurat_obj_subset.Robj: The Seurat R-object containing only the cells expressing a given gene above the threshold value. It can be passed to the next Seurat tool, or imported into R, but it is not viewable in Chipster.
  • subset_plot.pdf: This is a visual for the number of remaining cells in the subset
