Extract information from Seurat object

Description

With this tool you can extract information from the Seurat object. The tool will produce

  • a text file containing information of the different slots in the Seurat object
  • a table of the meta.data data frame in the Seurat object

    • Details

    This tool can be used to access the data stored in the Seurat object. Moreover, this tool can be used to check whether your scRNA-seq data can be used as the reference dataset in the spatially resolved trancriptomics tools: "Seurat v4 -Integration with scRNA-seq data" and "Seurat v4 and SCDC -Identify cell types using deconvolution".

    As input, you can give a Seurat R-object (Robj) or an RDS file containing scRNA-seq data.

    Output

    • slots.txt: Text file containing information of the different slots in the Seurat object
    • meta_data.tsv: Table of the meta.data data frame in the Seurat object
