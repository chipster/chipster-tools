Seurat v4 -Extract information from Seurat object

Description

With this tool you can extract information from Seurat object. The tool will produce a text file containing an overall view to the contents of the object. You can also choose to produce

  • a table of the genes in the object
  • a table of the highly variable genes in the object
  • a text file of the R commands that have been run to the object
  • a table of the metadata data frame
  • the tissue image stored in the object (in case you have spatial transcriptomics data)

    • Details

    The single-cell RNA-seq or spatially variable transcriptomics data is stored in the Seurat object in specific data slots. With this tool, you can easily access and extract information from the chosen slots.

    As input, you can give a Seurat R-object (Robj) or an RDS file containing scRNA-seq data or spatial transcriptomics data.

    Output

    • slots.txt: Text file containing information of the different slots in the Seurat object
    • gene.tsv: Table containig the genes in the active assay of the object
    • var_genes.tsv: Table containig the highly variable in the active assay of the object
    • commands.tsv: Table containing the R commands that have been run to the object
    • meta_data.tsv: Table of the meta.data data frame in the Seurat object
    • spatial_image.pdf: Pdf file containing the tissue image stored in the Seurat object
