Seurat -Visualise top expressed genes

Description

With this tool user can identify and visualise top expressed genes in the data. As input the user gives the Seurat R-object (.Robj) and the tool produces a boxplot of the most expressed genes. Code from https://nbisweden.github.io/workshop-scRNAseq/labs/compiled/seurat/seurat_07_spatial.html.

Details

Boxplot shows the total count per cell for each top expressed feature.

For more details, please check the Seurat tutorials.

Output
