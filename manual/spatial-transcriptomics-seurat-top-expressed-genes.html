Seurat -Visualise top expressed genes

Description

With this tool, the user can identify and visualise top expressed genes in the data. As input, the user can provide the Seurat R-object (.Robj) and the tool produces a boxplot of the most highly expressed genes.

Code can be found at this site: https://nbisweden.github.io/workshop-scRNAseq/labs/compiled/seurat/seurat_07_spatial.html

Details

Boxplot shows the total count per spot for the most highly expressed features.

For more details, please check the Seurat tutorials.
