Extract information from the Phyloseq object

Description

With this tool you can extract information from the Phyloseq object:

  • OTU or ASV table
  • taxonomy table
  • Dna sequences stored to refseq in the Phyloseq object in FASTA
  • Dna sequences stored to refseq in the Phyloseq object as tabular data
  • Sample information which is the Phenodata information

Parameters

  • Extract OTU table? (yes, no) [no]
  • Extract taxonomy table? (yes, no) [no]
  • Extract DNA sequences in FASTA? (yes, no) [no]
  • Extract DNA sequences as tabular data? (yes, no) [no]
  • Extract sample information? (yes, no) [no]

Details

This tool can be used to access the data/information saved to the Phyloseq object.
You can extract the information if it's stored to the Phyloseq object. You can find out what information it has from the summary .txt file produced when the object was created.
With this tool, one option is to get the full DNA sequences which were stored into the refseq() slot when the Phyloseq object was created from the ASV table. The full DNA sequences were changed to shorter strings in order to get the visualization looks nicer. In the refseq() slot is stored the information of which full DNA sequence correspond to ASV1, ASV2 and etc.
This information can be useful if you want to for example merge the found ASVs with other datasets or index into reference databases. The sequences can be saved either in FASTA format or as tabular data (.tsv file).

This tool can be used also to investigate how the Phenodata table, in other words the sample information merged to the Phyloseq object looks like or if you want to get the OTU/ASV table and the taxonomy table used to create the object.

Output

The analysis output consists of the following:
  • otu_table.tsv: OTU table from the Phyloseq object
  • taxonomy_table.tsv: Taxonomy table from the Phyloseq object.
  • dna_sequences.fasta: Full DNA sequences of the ASVs stored to the refseq() of the Phyloseq object in FASTA format
  • dna_sequences.tsv: Full DNA sequences of the ASVs stored to the refseq() of the Phyloseq object as tabular data
  • sample_information.tsv: The same tsv file as the phenodata table containing the sample information of the Phyloseq object
