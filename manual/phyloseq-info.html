EExtract Phyloseq information

Description

With this tool you can extract the DNA sequences stored to refseq, otu_table, taxonomy_table and sample_information from the given Phyloseq object.

Parameters

  • Do you want to extract the OTU table
  • Do you want to extract the taxonomy table
  • Do you want to extract the DNA sequences stored to refseq
  • Do you want to extract the sample infomartion

Details

This tool can be used to access the data/information saved to the Phyloseq information.

Output

The analysis output consists of the following:
  • otu_table.tsv: OTU table from the Phyloseq object
  • taxonomy_table.tsv: Taxonomy table from the Phyloseq object.
  • dna_sequences.tsv: Full DNA sequences of the ASVs stored to the refseq() of the Phyloseq object
  • sample_information.tsv: The same tsv file as the phenodata table containing the sample information of the Phyloseq object
