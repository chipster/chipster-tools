Seurat v5 -Identify spatially variable genes based on clusters

Description

Find differentially expressed genes (DEGs) between two clusters and visualize spatially variable genes.

Parameters

  • First cluster [1]
  • Second cluster [2]
  • Limit testing to genes which are expressed in at least this fraction of spots [0.01]
  • Limit testing to genes which show at least this fold difference [0.1]
  • Test for differential expression [wilcox]
  • Report only positive marker genes [FALSE]

Details

As an input, you can provide the Seurat object after clustering.

Select the clusters in which you would like to identify spatially variable genes. As an example, differential expression is performed between clusters 1 and 2. The first cluster is used as the identity class to define markers and the second cluster is used as the identity class for comparison.

Select the test used for differential expression. Options for this are the Wilcoxon Rank Sum Test (wilcox) or the hurdle model tailored to scRNA-seq analysis called MAST.

The FindMarkers() function finds DEGs between clusters. This function has a parameter for min.pct, or genes found in a minimum percentage of spots in the two clusters, where the default is set to 0.01. The function also includes the logfc.threshold parameter, which is a threshold where genes need to be at least a certain X-fold difference between the two groups of spots; the default is set to 0.1.

Output

  • Markerplot.pdf
