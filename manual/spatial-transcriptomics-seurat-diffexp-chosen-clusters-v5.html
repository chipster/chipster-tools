Seurat v5 -Identify spatially variable genes based on clusters

Description

This tool finds differentially expressed genes (DEGs) between two clusters and visualizes them on top of the tissue histology.

As an input, you can provide the Seurat object after the "Seurat v5 -Clustering" step.

Parameters

  • First cluster [1]
  • Second cluster [2]
  • Limit testing to genes which are expressed in at least this fraction of spots [0.01]
  • Limit testing to genes which show at least this fold difference [0.1]
  • Test for differential expression [wilcox]
  • Report only positive marker genes [FALSE]
  • Determine color scale based on all genes [no]

Details

The user can select the clusters where differentially expressed genes would like to be identified. The tool identifies positive and negative marker genes for the clusters of interest, determined by the user. You can choose to return only positive marker genes by selecting "TRUE" as the parameter value for "Report only positive marker genes". You can also select the test used for differential expression testing using the parameter "Test for differential expression" from two options: Wilcoxon Rank Sum Test (wilcox) or the hurdle model tailored to scRNA-seq analysis called MAST.

The user can also filter out genes prior to statistical testing by requiring that a gene has to be expressed in at least a certain fraction of spots in either of the two clusters using the parameter "Limit testing to genes which are expressed in at least this fraction of spots". You can also filter out genes whose log2 fold change between the two clusters is less than a given value using the parameter "Limit testing to genes which show at least this fold difference". Both of these parameters can be set to 0, but with a significant increase in computation time since this will test a large number of genes that are unlikely to be highly expressed.

This tool finally visualizes and lists the discovered DEGs between clusters. By default, the color scale is determined for each gene individually and may differ between genes. If you wish to compare expression between different genes, it may be useful to set the parameter value for "Determine color scale based on all genes" to "yes" so that the color scale is the same for all genes.

For more details on the spatial transcriptomics analysis pipeline, please check the Seurat tutorial.

Output

  • Markerplot.pdf
  • spatially_variable_genes.tsv
