Annotate cells with UCell

Description

With this tool, you can annotate celltypes for cells/clusters with your own marker genes and cell types.

Parameters

  • Cell types to plot [T cells, B cells, NK cells, Monocytes]
  • Gene sets for cell types [CD3D, CD3E, IL7R, sep, MS4A1, CD79A, CD79B, sep, NKG7, GNLY, PRF1, sep, CD14, LST1, S100A8]
  • width of plots [10]
  • height of plots [10]
  • Point size in tSNE and UMAP plots [1]
  • Label size in the output plots [4]

Details

With this tool, you can annotate cells with UCell. UCell is an R package for scoring gene signatures in single-cell datasets. UCell scores, based on the Mann-Whitney U statistic, are robust to dataset size and heterogeneity, and their calculation demands relatively less computing time and memory than other robust methods, enabling the processing of large datasets (>10^5 cells). UCell can be applied to any cell vs. gene data matrix, and includes functions to directly interact with Seurat and Bioconductor's SingleCellExperiment objects.

UCell does not have a built-in database of marker genes or cell types. You need to firstly name cell types yourself and then input the marker genes resepective to each cell type using "sep" as a separator between gene sets.

UCell article: UCell: Robust and scalable single-cell gene signature scoring https://www.sciencedirect.com/science/article/pii/S2001037021002816

Output

  • seurat_obj_clustering.Robj: The Seurat R-object to pass to the next Seurat tool, or to import to R. You cannot open this file in Chipster, but you can view the contents of the object using the Extract information from Seurat object tool.
  • Plots.pdf: UMAP of cell type scores in cells and clusters, assigned cell types per cell and cluster with labels.
