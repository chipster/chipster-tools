Filter by taxonomic group

Description

Tidies a phyloseq object so that OTUs only from the desired taxonomic group (bacteria, archaea, eukaryotes or fungi) are retained. For bacteria, archaea and eukaryotes, filtering is performed at the domain level (Bacteria, Archaea, Eukaryota). For fungi, filtering is performed at the kingdom level (Fungi). Features with ambiguous phylum-level annotation (e.g. NA, unknown, uncharacterized) are removed. Produces a phylum-level taxonomy summary and prevalence table following filtering, and saves the resulting phyloseq object as an Rda file.

Parameters

  • Taxonomic group to be retained (bacteria, archaea, eukaryotes or fungi; the default is bacteria)

Details

The tool requires the following input files:

  • A phyloseq object in Rda format
  • Data groupings are automatically fetched from a phenodata file linked to a Mothur shared file (file.opti_mcc.shared)
The tool employs the following R functions and packages:
  • subset_taxa(), tax_table() and taxa_sums() (phyloseq)
  • ddply() (plyr)
  • rename() (dplyr)

Output

The tool produces the following output files:

  • Phyloseq object following removal of undesired features (ps_xxxx.Rda; possible file names are ps_bacteria.Rda, ps_archaea.Rda, ps_eukaryota.Rda and ps_fungi.Rda)
  • Text summary with phylum frequencies and prevalences following filtration (ps_xxxx_taxon.txt; possible file names are ps_bacteria_taxon.txt, ps_archaea_taxon.txt, ps_eukaryota_taxon.txt and ps_fungi_taxon.txt)
