Seurat v5 -Visualize clusters

Description

With this tool, the user can visualize selected clusters separately on top of the tissue image to distinguish the spatial localization of individual clusters. The user can provide a Seurat (.Robj) as input and select which clusters to plot (e.g., 1,2,3,5).

Please note that if you list multiple clusters, use a comma (,) as a separator.

Parameters

  • Clusters to plot [2,1]

Details

The plot shows the selected clusters on top of the tissue image.

For more details on the spatial transcriptomics analysis pipeline, please check the Seurat tutorial.

Output

  • spatialdimplot.pdf
