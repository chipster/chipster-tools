Remove OTUs with 0-2 occurrences

Description

Prunes a phyloseq object so that only OTUs with >0, >1 or >2 occurrences are retained. The resulting object is saved in Rda format.

Parameters

  • Filtering threshold (>0, >1 or >2 occurrences; the default is >2)

Details

The tool requires the following input files:

  • A phyloseq object in Rda format
  • Data groupings are automatically fetched from a phenodata file linked to a Mothur shared file (file.opti_mcc.shared)
The tool employs the following R functions and packages:
  • prune_taxa() (phyloseq)

Output

The tool produces the following output files:

  • Phyloseq object following pruning step (ps_pruned.Rda)
  • Text summary of phyloseq object following pruning step (ps_pruned.txt)
