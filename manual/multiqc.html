Combine reports using MultiQC

Description

This tool combines reports from supported tools such as FastQC.

Parameters

Details

MultiQC is a tool that can summarize analysis results for multiple tools and samples in a single report.

Some Chipster tools can produce a mqc file in addition to the primary result files. Select the mqc files for all the tools and samples you want to include in the report and run the tool.

Currently supported tools

  • Quality control / Read quality with FastQC

More tools will be added later.

Output

An html report with plots and tables produced by the analysis modules. Please note that you have to select the visualization option "Open in external browser".

Reference

This tool is based on the MultiQC package by Philip Ewels et al.
