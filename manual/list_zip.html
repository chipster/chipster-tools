Utilities / List contents of a zip file

Description

Lists the contents of a zip file.

Parameters

  • None

Details

The tool lists the contents of a zip file into a file.

This list can be useful e.g. when wanting to extract only some files from a zip file. You can export it, edit it, add it back to Chipster and use it as list file with tool "Utilities / Extract .zip file".

Output

The output is a text file.
