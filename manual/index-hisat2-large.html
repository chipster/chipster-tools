Create Hisat2 index for large genomes

Description

This tool creates index files for the Hisat2 aligner.

Parameters

None

Input

  • input.fa: reference genome fasta file
  • input.gtf: reference genome gtf file

Output

This tool produces multiple files that together form the index of the given genome.

Reference

This tool is based on the HISAT2 package. Please cite the following article: Kim D, Langmead B and Salzberg SL. HISAT: a fast spliced aligner with low memory requirements Nature Methods 2015.
