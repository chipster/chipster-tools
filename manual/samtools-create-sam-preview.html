Utilities / Create a preview for BAM

Description

Creates a SAM format preview for a BAM file.

Parameters

None

Details

Creates a SAM format preview for a BAM file. The preview contains the header and 200 first alignments.

Output

Output is a SAM file

References

This tool is based on the SAMtools package.
