Create a preview for BAM

Description

Creates a SAM formated preview for a BAM file.

Parameters

None

Details

The preview contains the header lines and the first 200 alignment lines. For information about the SAM header and content, please see the SAM format specification.

Output

Output is a SAM file.

References
