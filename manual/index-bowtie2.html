Create Bowtie2 index

Description

This tool creates index files for the Bowtie2 aligner.

Parameters

None

Input

The genome's fasta file is taken as an input.

Output

This tool produces multiple files that together form the index of the given genome.

Reference

Langmead B, Salzberg S. Fast gapped-read alignment with Bowtie 2. Nature Methods. 2012, 9:357-359.
