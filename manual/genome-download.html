Download genome from Ensembl

Description

Download genome fasta and gtf files from Ensembl. The files are downloaded from the Ensembl FTP server. Verteberates are downloaded from ftp.ensembl.org other species are downloaded from ftp.ensemblgenomes.org.

Parameters

  • Species [drosophila_melanogaster]

    • Like written in the directory listing for verteberates, fungi, plants or in one of the bacteria collections. Omit the trailing forward slash character "/".

  • Version [BDGP6.32]

    • To find the genome assembly version, use the above links to find your species, open it, and then open a folder called "dna". For example, for the fruit fly this would be http://ftp.ensembl.org/pub/current_fasta/drosophila_melanogaster/dna/. Locate a fasta file package from the file listing, which ends with ".dna.toplevel.fa.gz". In this case it is "Drosophila_melanogaster.BDGP6.46.dna.toplevel.fa.gz". It starts with a species name (but now the first letter is in uppercase), followed by a period "." and ends with ".dna.toplevel.fa.gz". In the middle you can find the genome version in the correct format: "BDGP6.32".

  • Ensembl release [current]

    • Use current to use the latest Ensembl release. If you want to download older version, go the species' Ensembl page. The dropdown menu on the left lists all the different genome versions and their latest Ensembl releases. For example, To download the BDGP6.22 assembly of Drosophila melanogaster, set this parameter to release-98.

  • Ensembl site (verteberates, fungi, plants, bacteria) [verteberates]

    • Check the address of the file listing. If it starts with ensembl.org, select "verteberates". If if starts ensemblgenomes.org, select "fungi", "plants" or "bacteria".

  • Action (Check version, Download) [Download]

    • The tool can be used in the two modes. The mode Download finds the requested genome from the server and downloads it. The other mode Check version only checks that the requested genome version is found from the server and then the tool finishes successfully or with an error accordingly. This is mostly useful for external scripts that want to check the versions before starting the slow download.

  • Preferred assembly (Toplevel, Primary assembly) [Primary assembly]

    • Toplevel file includes chromsomes, regions not assembled into chromosomes and N padded haplotype/patch regions. Primary assembly contains all toplevel sequence regions excluding haplotypes and patches. When Primary assembly is not available, the tool will automatically fall back to the Toplevel assembly.

Input

None

Output

  • output.fa: reference genome fasta
  • output.gtf: reference genome gtf

Reference

This tools downloads data from Ensembl. Please see their page of Scientific Publications for how to cite.
