Overview of taxon composition

Description

Tabulates the overall OTU count and a summary of taxon frequencies in a phyloseq object.

Parameters

  • Level of biological organization for tabulating taxon composition (Phylum, Class, Order, Family, Genus, Species; the default is Phylum)

Details

The tool requires the following input files:

  • A phyloseq object in Rda format
  • Data groupings are automatically fetched from a phenodata file linked to a Mothur shared file (file.opti_mcc.shared)
The tool employs the following R functions and packages:
  • ntaxa() and tax_table() (phyloseq)

Output

The tool produces the following output files:

  • Text summary of the overall OTU count and taxon frequencies (ps_taxon.txt)
