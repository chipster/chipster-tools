Filter out non-bacterial features

Description

Tidies a phyloseq object so that only bacterial OTUs are retained. Filters out OTUs that have not been classified as Bacteria (kingdom level), that have been classified as NA at the phylum level, and/or have been classified as chloroplast sequences at the class level. Produces a phylum-level taxonomy summary and prevalence table following filtering, and saves the resulting object as an Rda file.

Parameters

None

Details

The tool requires the following input files:

  • A phyloseq object in Rda format
  • Data groupings are automatically fetched from a phenodata file linked to a Mothur shared file (file.opti_mcc.shared)
The tool employs the following R functions and packages:
  • subset_taxa(), tax_table() and taxa_sums() (phyloseq)
  • ddply() (plyr)
  • rename() (dplyr)

Output

The tool produces the following output files:

  • Phyloseq object following removal of non-bacterial features (ps_bacteria.Rda)
  • Text summary with phylum frequencies and prevalences following filtration (ps_bacteria_taxon.txt)
