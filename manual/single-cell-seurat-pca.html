Seurat -PCA

Description

This tool performs principal component analysis (PCA) on the highly variable genes across the single cells selected using the tool Filtering, regression and detection of variable genes.

Parameters

  • Number of principal components (PCs) to plot as heatmaps [12]
  • Number of PCs to compute [20]
  • Print loadings in a file [no]
  • Number of genes to list in the loadings file [5]

Details

Please input the Seurat R-object (Robj) that was generated using the tool Seurat -Filtering, regression and detection of variable genes.

If you have a low number of cells in your data, you may receive an error message. In this case, try lowering the number of PCs to compute.

To overcome extensive technical noise in any single gene for scRNA-seq data, Seurat clusters cells based on their PCA scores. It does so by having each PC represent a 'metagene' combining information across a correlated gene set. Therefore, even though determining the number of PCs to include is an important step, it can also be challenging. The result file PCAplots.pdf contains several plots showing the primary sources of heterogeneity in the dataset, as well as provides a basis for deciding the number of PCs to include downstream.

In the first plot, the top genes associated with the first two PCs are shown. The second plot shows the PCA plot: PCs 1 and 2 represent the cells in the dataset, and the third plot is a heatmap focusing on PC 1.

In the heatmap, both cells and genes are sorted by their PC scores: both extremes are shown. Rows represent 'extreme' genes, and columns are 'extreme' cells. They are 'extreme' in their PCA scores, so they are the genes which best determine the particular PC, i.e., the separation between cells. Similarly, cells that are 'furthest away' (in the yellow or purple end) from each other are on the PC 1 spectrum. Purple: positive correlation, yellow: negative correlation. The heatmap may be useful to cross-check with the PC. For example, if the genes seem to be related to the cell cycle phase, one may want to consider regressing out that effect, or at least it is good to acknowledge this.

Next, plots can be used to decide which PCs to use. By default, 12 heatmaps are plotted, focusing on the first 12 PCs, in which both cells and genes are sorted by their PC scores. You can change the number of heatmaps to plot. Please explore the PCs for downstream analysis: heatmaps can display the 'extremes' across both genes and cells, and it can be useful to exclude PCs that may be driven primarily by ribosomal, mitochondrial, or cell cycle genes.

The last plot shows the standard deviations (SDs) of the PCs. The cutoff can be identified by the 'elbow' in the graph.

You can also use the tool to output a *.txt file listing the top N (default = 5) genes associated with high and low loadings for the first 5 PCs in the PCAloadings.txt file.

For more details, please refer to the Seurat tutorials.

Output

  • seurat_obj.Robj: The Seurat R-object to pass to the next Seurat tool, or to import into R, but not viewable in Chipster.
  • PCAplots.pdf: Plots presenting the PCs. The last plots are meant to aid in the selection of PCs for downstream analysis.
  • PCAloadings.txt: If selected, the loadings (=genes responsible) of the PCs.
