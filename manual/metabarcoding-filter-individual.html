Remove selected phyla

Description

Removes up to five user-specified phyla from a phyloseq object. Produces a taxonomy summary at the phylum level and saves the resulting object as an Rda file.

Parameters

  • Phyla to be removed (one is required; up to five can be removed)

Names of phyla to be removed are manually typed using the text field provided.

Details

The tool requires the following input files:

  • A phyloseq object in Rda format
  • Data groupings are automatically fetched from a phenodata file linked to a Mothur shared file (file.opti_mcc.shared)
The tool employs the following R functions and packages:
  • subset_taxa() and tax_table() (phyloseq)

Output

The tool produces the following output files:

  • Phyloseq object following removal of individual phyla (ps_ind.Rda)
  • Text summary of taxon frequencies following filtration (ps_ind_taxon.txt)
