Remove selected taxa

Description

This tool can be used to remove chloroplast and mitochondrial sequences from a phyloseq object, and up to five user-specified taxa at the desired level of biological organization. Produces a taxonomy summary of the resulting phyloseq object and saves it as an Rda file.

Parameters

    Remove class Chloroplast (yes, no; default is yes)
      Remove family Mitochondria (yes, no; default is yes)
    • Level of biological organization for manual taxon removal (Phylum, Class, Order, Family, Genus or Species; default is Phylum)
    • Taxa to be removed (up to five can be specified)

    For manual taxon removal (removal of taxa other than Chloroplast and Mitochondria), the names of the taxa are typed into the text field provided.

    Details

    The tool requires the following input files:

    • A phyloseq object in Rda format
    • Data groupings are automatically fetched from a phenodata file linked to a Mothur shared file (file.opti_mcc.shared)
    The tool employs the following R functions and packages:
    • subset_taxa() and tax_table() (phyloseq)

    Output

    The tool produces the following output files:

    • Phyloseq object following removal of individual taxa (ps_ind.Rda)
    • Text summary of taxon frequencies following filtration (ps_ind_taxon.txt)
