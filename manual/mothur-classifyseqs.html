Classify 16S or 18S sequences to taxonomic units using Silva

Description

Given a fasta file of aligned sequences and a count file generated by Mothur, this tool assigns the sequences to taxonomic units (~ species).

Parameters

  • Reference (silva.nr_v138.1, own) [silva.nr_v138.1]
  • Number of iterations (10-1000) [100]

Details

This tools assigns aligned sequences to taxonomic units using the Wang method and a reference.
  • As of the 138 release, SILVA databases, taxonomy and all the files provided for download are freely available for academic and commercial use: SILVA-license
  • If you use your own reference, please provide the fasta and taxonomy file as input.

The bootstrap confidence score is calculated with 100 iterations by default. This low number of iterations is used as a default to make the tool faster, please consider using higher number (1000 for example).

If you would like to remove certain taxa or filter by a taxonomic group, you can use the following Phyloseq tools in Chipster (note that prior to this you need to run the tools Generate input files for phyloseq and Convert mothur files into phyloseq object):

  • Filter by taxonomic group tidies a phyloseq object so that OTUs only from the desired taxonomic group (bacteria, archaea, eukaryotes or fungi) are retained. Features with ambiguous phylum-level annotation (e.g. NA, unknown, uncharacterized) are removed.
  • Remove selected taxa removes chloroplast and mitochondrial sequences from a phyloseq object, and up to five user-specified taxa at the desired level of biological organization.

This tool is based on the classify.seqs command of the Mothur package.

Output

The analysis output consists of the following files:

  • sequences-taxonomy-assignment.txt: Taxonomy assignment for each sequence
  • classification-summary.tsv: Taxonomic outline indicating the number of sequences that were found at each level
