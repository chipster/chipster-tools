Extract UMIs from QuantSeq reads

Description

Given a Lexogen QuantSeq FASTQ file, this tool removes the TATA spacer and extracts the 6-base unique molecular identifier (UMI) and stores it in read's name.

Parameters

  • None

Details


QuantSeq FWD-UMI libraries contain a 6 nt UMI at the start of the read, followed by a 4 nt spacer that separates the UMI from the random priming sequence that initiates second strand synthesis. The spacer has the same sequence (TATA) in all reads, which generates low diversity signals for cycles 7-10 of Read 1. This low diversity signature is visible in FastQC analysis results.

This tool removes the TATA spacer and extracts the 6-base UMI and stores it in read's name.

Output

The output consists of the following files:

  • FASTQ file containing the trimmed reads
  • log.txt file showing how many reads contained the TATA motif in position 7-10 and were thus kept and processed
