Merge phenodata to the Phyloseq object

Description

Given a Phyloseq object saved as a .Rda file and a Phenodata table, this tool merges the sample information from the Phenodata table to the Phyloseq object.

Parameters

  • Phenodata variable with unique IDs for each community profile

Details

This tool merges the sample information from the phenodata table to the phyloseq object with the merge_phyloseq() command.
After the DADA2/asv pipeline, the sample information needs to be merged to the phyloseq object before the next analyse tools in Chipster.
After the Mothur/OTU pipeline, the phenodata.tsv is already merged to the Phyloseq object but this tool can be used to update the sample information.

Output

The analysis output consists of the following:
  • ps.Rda: A Phyloseq object saved as a .Rda object.
  • ps_sample_summary.txt: Text summary of the phyloseq object.
