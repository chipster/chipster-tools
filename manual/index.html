Chipster manual

Chipster is an easy-to-use analysis software for high-throughput data. There are currently two interfaces: a Web App and a Java client. The latter requires Java Web Start, and it will be phased out by 15 Nov 2020. Both interfaces offer the same comprehensive collection of analysis tools for

  • RNA-seq
  • Single cell RNA-seq
  • Small RNA-seq (miRNA and VirusDetect)
  • Community analysis of 16S data
  • ChIP-seq and DNase-seq
  • Exome and whole genome seq (variants and copy number aberrations)
  • Microarrays
