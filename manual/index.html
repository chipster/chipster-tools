Chipster manual

Chipster is an easy-to-use analysis software for high-throughput data. It offers a comprehensive collection of analysis tools for

  • Single-cell RNA-seq
  • Spatially resolved transcriptomics (Visium)
  • RNA-seq
  • Small RNA-seq (miRNA and VirusDetect)
  • Microbial community analysis (16S, 18S and ITS)
  • ChIP-seq and DNase-seq
  • Exome and whole genome seq (variants and copy number aberrations)
  • Microarrays

You can get started by watching this video, where you will learn

  • what Chipster is
  • how the user interface works
  • how to import data
  • how to store and share your analysis
  • how to run many analysis jobs at the same time
  • how to get help
  • where to find training materials
