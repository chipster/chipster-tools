Seurat -Visualize genes with cell type-specific responses in multiple samples

Description

This tool generates plots based on your selection of markers or genes across conditions. This tool can be used on Seurat objects containing two or more samples.

Parameters

  • Markers to plot [CD3D, CREM, HSPH1, SELL, GIMAP5]

Details

Please input a combined Seurat object.

Select the markers you would like to inspect by listing them in the parameter row. The markers can also be input in a text file format. Separate the names with comma(s) (,). As an example, the parameter is set to markers "CD3D, CREM, HSPH1, SELL, GIMAP5".

The "SplitDotPlotGG" can be useful for visualizing conserved cell type markers across conditions, showing both the expression level and percentage of cells in a cluster expressing any given gene. The size of the dot encodes the percentage of cells within a class (=cluster), while the color encodes the AverageExpression level of 'expressing' cells (stronger color, more expression).

Another useful way to visualize the changes in gene expression is using the FeatureHeatmap function. This will display FeaturePlots of the gene list, split according to condition.

In case of more than two samples, the plots are divided into multiple pages.

For more details, please refer to the Seurat tutorials for multiple sample analysis.

Output

  • split_dot_plot.pdf: Plots showing user-defined markers or genes across conditions
