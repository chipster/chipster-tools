Seurat v4 -Subset out additional cells

Description

This tool subsets out anatomical regions by removing additional cells to help with the integration with scRNA-seq data. Then it visualises the results with SpatialDimPlot.

Parameters

  • Rows to subset [400]
  • Columns to subset [150]

Details

As input, you can give the Seurat object from the tool "Seurat v4 -Subset out additional clusters."

Select the cells you want to subset, by giving rows and columns to be removed.

Output

  • seurat_obj_subset2.Robj: Can be passed to the next Seurat tool, or imported to R. Not viewable in Chipster.
  • subset_plot.pdf
