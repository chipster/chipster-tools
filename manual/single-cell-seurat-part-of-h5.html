Seurat -Extract only gene expression data from a multiomics h5 object

Description

For a given cluster, this tool gives you the differentially expressed genes between the sample groups listed by the user. Sample groups are determined in the Setup tool with the Sample group parameter.

Parameters

  • Output file name

Details

As input, give a multiomics h5 object, from which you want to extract the Gene Expression part only. The output will be a matrix as a .tsv table, that can then be passed on to the Setup tool, or the Separate samples tool.

For more details, please check the Seurat tutorials for multiple sample analysis.

Output

  • sample_name.tsv files: Matrix with the Gene Expression data from the input h5 object.
