Subset FASTQ

Description

Returns a subsample of N reads from a FASTQ file.

Parameters

  • Size of subset (Integer) [100000]
  • Random Seed (Integer) [11]

Details

Returns a subsample of N reads from a FASTQ file.

The input can be a single FASTQ file or a tar package containing multiple FASTQ files.

When using paired-end data, use same random seed for both files to keep pairing.

Output

The output is a gzipped FASTQ file or a tar file containing gzipped FASTQ files.

References
