PERMDISP for OTU abundance data

Description

Performs a permutation-based test of the multivariate homogeneity of group dispersions (PERMDISP, 999 iterations), using a distance matrix derived from OTU abundance data. Accepts a single phenodata variable at a time, with the option to run three consecutive tests (for three different variables).

Parameters

  • Phenodata variable(s) used to specify the formula to be tested (up to three variables can be specified)

Details

The tool requires the following input files:

  • An Rda file including phyloseq object sample data and a distance matrix (ps_dist.Rda, produced using the tool "Distance matrices and ordinations")
  • Data groupings are automatically fetched from a phenodata file linked to a Mothur shared file (file.opti_mcc.shared)
The tool employs the following R functions and packages:
  • betadisper() and permutest() (vegan)

Output

The tool produces the following output files:

  • Text summary of PERMDISP output (permdisp.txt)
  • Rda file with dispersion and test data (ps_disp.Rda)
If the PERMDISP result is statistically significant (p < 0.05), the Rda file produced by this tool (ps_disp.Rda) can be used as the input for the "Post-hoc pairwise Tukey's HSD for OTU abundance data" tool.
