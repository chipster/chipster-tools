Filter fasta

Description

Keep only primary chromosomes in fasta. Chromosomes that have karyotype data in Ensembl REST API are considered as the primary chromosomes.

Parameters

None

Input

  • input.fa: reference genome fasta file

Output

  • output.fa: reference genome with only primary chromosomes

Reference

This tools is based on Ensembl REST API and SAMtools package. Please cite the following articles:
