Additional prevalence summaries

Description

Produces prevalence summaries using phyloseq to assist with data preparation for downstream analyses. Plots total read counts against prevalence values for each phylum. Also tabulates the overall numbers of features with zero reads, one read (singletons) and two reads (doubletons).

Parameters

None

Details

The tool requires the following input files:

  • A phyloseq object in Rda format
  • Data groupings are automatically fetched from a phenodata file linked to a Mothur shared file (file.opti_mcc.shared)
The tool employs the following R functions and packages:
  • fast_melt() (available on the phyloseq GitHub repository)
  • otu_table() (phyloseq)
  • ggplot() (ggplot2)
For details on the steps used by this tool, see Callahan et al. (2016).

Output

The tool produces the following output files:

  • Prevalence plot with 5% filtering threshold displayed as an example (ps_prevalence.pdf)
  • Text summary of features with low prevalence and total counts (ps_low.txt)
