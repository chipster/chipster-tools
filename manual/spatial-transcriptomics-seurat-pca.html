Seurat -PCA, clustering, and visualisation

Description

This tool performs principal component analysis (PCA) on the highly variable genes, selected using the tool Seurat v4 -Filter spots, normalize with SCTransform and detect high-variance genes.

Parameters

  • Number of PCs to compute [30]
  • Print loadings in a file [no]
  • Number of genes to list in the loadings file [5]

Details

As input, give a Seurat R-object (Robj).

If you have low number of spots in your data, you might get an error message. Try lowering the number of principal components to compute.

This tool produces the following cluster plots.

  • Dimplot showing the results of the dimensional reduction on a 2D scatter plot where each point is a spot and it's positioned based on the spot embeddings determined by the reduction technique.
  • SpatialDimPlot showing the resulting clusters overlaid on the tissue image.

You can also request the tool to output a txt file which lists the top N (default = 5) genes associated with high/low loadings for the first 5 PCs in the PCAloadings.txt file.

For more details, please check the Seurat tutorials.

Output

  • seurat_obj_pca_clust.Robj: The Seurat R-object to pass to the next Seurat tool, or to import to R. Not viewable in Chipster.
  • UMAP_plot.pdf: Graphics presenting the principal components. Last plots are meant to aid in the selection of the PCs for the downstream analysis.
  • PCAloadings.txt: If requested, the loadings (=genes responsible) of principal components
