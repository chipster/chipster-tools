Seurat -PCA, clustering, and visualisation

Description

This tool performs principal component analysis (PCA) on the highly variable genes, which were previously selected using the tool Seurat v4 -Filter spots, normalize with SCTransform and detect high-variance genes.

Parameters

  • Number of principal components (PCs) to compute [30]
  • Print loadings in a file [no]
  • Number of genes to list in the loadings file [5]

Details

Provide a Seurat R-object (Robj) as input.

If you have few spots in your data, you may get an error message. Try lowering the number of PCs to compute.

This tool produces the following cluster plots:

  • Dimplot, shows the results of the dimensional reduction on a 2D scatterplot where each point is a spot positioned based on the spot embeddings determined by the reduction technique.
  • SpatialDimPlot, shows the resulting clusters overlaid on the tissue image.

You can also use the tool to output a *.txt file. This file will list the top genes (default = 5) associated with high or low loadings for the first 5 PCs in the PCAloadings.txt file.

For more details, please check the Seurat tutorials.

Output

  • seurat_obj_pca_clust.Robj: The Seurat R-object to pass to the next Seurat tool, or to import into R, but not viewable in Chipster.
  • UMAP_plot.pdf: A plot presenting the PCs. Last plots are meant to aid in the selection of the PCs for the downstream analysis.
  • PCAloadings.txt: If selected, the loadings (=genes responsible) of PCs
