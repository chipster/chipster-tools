Utilities / Split table

Description

Splits a TSV format table into several files.

Parameters

  • Substitute existing header with "identifier" and "sample" (yes, no) [no]

Details

Tool splits TSV (tab separated columns/values) format table into several files.
All files share the first column of the input table, typically an identifier.
For each additional column a file is created containing the first column of the original file and the single additional column.

If option substitute existing header is set to "yes":
- The column title of the first column is set to "identifier".
- The column title of the additional columns are named to "sample".
- The original column titles of the additional columns are part of the new file names.
This is for example needed for Microarray -> Normalisation -> Illumina.

Output

  • split{...}{_colTitle}.tsv: The splitted tsv files, starting with split02.tsv, containing column1(identifier) and column2(sample)
