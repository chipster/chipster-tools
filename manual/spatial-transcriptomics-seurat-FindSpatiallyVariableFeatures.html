Seurat v4- Find spatially variable features

Description

This tool is an alternative to Seurat v4- Identification of spatially variable features. It searches features that exhibi spatial patterning in the absence of pre-annotation

Parameters

  • Maximum transparency [1]
  • Print loadings in a file [no]

Details

As inputs, give the Seurat object.

Select the cluster you want to inspect by setting its name in the parameter field. As an example, the parameter is set to cluster "3".

For more details, please check the Seurat tutorials for multiple sample analysis.

Output

  • conserved_markers.tsv: Conserved cell type markers across the conditions
