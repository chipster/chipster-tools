Seurat -Visualise gene expression

Description

With this tool user can visualise selected genes with SpatialFeaturePlot. As input the user gives the Seurat R-object (.Robj) and the name of the gene of interest (for example Hpca, Ttr...).
Note: If you list multiple gene names, use comma (,) as separator.

Parameters

  • Gene name ["Hpca, Ttr"]
  • Point size [1.6]
  • Minimum transparency of the points [1]
  • Maximum transparency of the points [1]

Details

Spatial feature plot shows molecular data overlaid on top of tissue histology. Good marker genes will show strongly in the histology image.

For more details, please check the Seurat tutorials.

Output

  • feature_plot.pdf
