Seurat -Visualise gene expression

Description

With this tool, the user can visualise selected genes using SpatialFeaturePlot. As input, the user can provide the Seurat R-object (.Robj) and the name of the gene(s) of interest (e.g., Hpca, Ttr).
Please note: Genes are case-sensitive, so the first letter needs to be capitalized and the remaining letters should be lowercase. If you list multiple gene names, use a comma (,) as a separator.

Parameters

  • Gene name ["Hpca, Ttr"]
  • Point size [1.6]
  • Minimum transparency of the points [1]
  • Maximum transparency of the points [1]

Details

Spatial feature plot shows molecular data overlaid on top of the histological tissue. Highly expressed marker genes will be clearly displayed in the histological image.

For more details, please check the Seurat tutorials.

Output

  • feature_plot.pdf
