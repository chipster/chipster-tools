Seurat v5 -Identify top expressed genes

Description

With this tool, the user can identify and visualize top expressed genes in the data. As input, the user can provide the Seurat object (.Robj) and the tool produces a boxplot of the most highly expressed genes.

Details

The boxplot shows the total UMI count per spot for the most highly expressed genes.

More information can be found from the spatial transciptomics tutorial.

Output
