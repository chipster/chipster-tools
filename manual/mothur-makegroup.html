Combine FASTQ or FASTA files and make a group file

Description

Combines FASTQ or FASTA files of each sample to one file and creates a Mothur group file. Input can be one or more FASTQ or FASTA files or a tar file. Input files can be gzipped.

Parameters

Details

For the Mothur-based tools in Chipster your samples need to be in one FASTA file. In order to assign sequences to a specific sample, you need also a Mothur group file. If you have single end data (e.g. Ion Torrent) and your samples are in separate FASTQ files, you can convert them FASTA and combine them with this tool, which also generates the group file.

Input can be one or more FASTQ or FASTA files or a tar file containing FASTQ or FASTA files.

This tool is based on the Mothur tools make.group, merge.files and summary.seqs.

Output

The output consists of the following files:

  • sequences.fasta.gz: Compressed FASTA file containing sequences from all the samples
  • sequences.groups: Mothur group file indicating what sequence belongs to what sample
  • sequences-summary.tsv: Summary statistics for the sequences
