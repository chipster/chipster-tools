Combine FASTA files and make a group file

Description

Combines FASTA files of each sample to one file and creates a groups file. FASTA files can be gzipped.

Parameters

None

Details

For the Mothur-based tools in Chipster your samples need to be in one FASTA file. In order to assign sequences to a specific sample, and you need a Mothur group file. If you have single end data (e.g. IonTorrent) and your samples are in separate FASTQ files, you can convert them FASTA, and then combine them with this tool, which also generates the group file.

This tool is based on the make.group and merge.files commands of the Mothur package.

Output

The output consists of the files:

  • sequences.fasta.gz: Compressed FASTA file containing all the sequences
  • sequences.groups: Mothur group file indicating what sequence belongs to what sample
