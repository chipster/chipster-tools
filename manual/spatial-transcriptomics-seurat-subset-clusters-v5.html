Seurat v5 -Subset out anatomical regions based on clusters

Description

This tool filters out anatomical regions based on user-selected clusters such that the analysis can be continued with only the subsetted data. This tool returns the subsetted Seurat object and visualizes the subsetted data.

Parameters

  • Subset of clusters [1,2,3,4,5]

Details

As an input, you can provide the Seurat object after the "Seurat v5 -Clustering" step.

For more details on the spatial transcriptomics analysis pipeline, please check the Seurat tutorial.

Output

  • seurat_obj_subset.Robj: The Seurat R-object to pass to the next Seurat tool, or to import into R. You cannot open this file in Chipster but you can view the contents of the object using the Seurat v5 -Extract information from Seurat object tool.
  • subset.pdf: Visualization of the subsetted clusters.
