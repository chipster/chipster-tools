Seurat v5 -Subset out anatomical regions based on clusters

Description

This tool filters out anatomical regions based on clusters to focus on a subset of data. The subsetted object is then visualized using a SpatialDimPlot.

Parameters

  • Subset of clusters [1,2,3,4,5]
  • Number of variable genes to return in SCTransform [3000]
  • Number of PCs to compute in PCA [50]

Details

Provide a Seurat R-object (Robj) after clustering as input.

For more details, please check the Seurat tutorial.

Output

  • seurat_obj_subset.Robj: The Seurat R-object to pass to the next Seurat tool, or to import into R. You cannot open this file in Chipster but you can view the contents of the object using the Seurat v5 -Extract information from Seurat object tool.
  • subset.pdf: Visualization of the clusters in a SpatialDimPlot.
