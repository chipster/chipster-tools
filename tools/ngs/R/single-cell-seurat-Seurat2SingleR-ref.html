Parameters

  • Tissue type [AUTO]

Details

With this tool, you can annotate cells with ScType. ScType a computational method for automated selection of marker genes based merely on scRNA-seq data. You can also get a graph showing the relationship of the clusters and the cell types, and a barplot showing the auto-detection of tissue type based on the expression of marker genes

Ianevski, A., Giri, A.K. & Aittokallio, T. Fully-automated and ultra-fast cell-type identification using specific marker combinations from single-cell transcriptomic data. Nat Commun 13, 1246 (2022). https://doi.org/10.1038/s41467-022-28803-w

For more details, please check:
The ScType GitHub repository

Output

  • seurat_obj_clustering.Robj: The Seurat R-object to pass to the next Seurat tool, or to import to R. You cannot open this file in Chipster, but you can view the contents of the object using the Extract information from Seurat object tool.
  • Plots.pdf: Plots showing auto detection tissue score if tissuetype is AUTO. UMAPs of annotated cell types and clusters. (EXTRA): If ScType does not find any duplicates, it also produces bubble plots.
